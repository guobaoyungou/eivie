## 常见问题排查

### 问题 1: 看不到新菜单
**原因**: 浏览器缓存  
**解决**: 强制刷新 `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac)

### 问题 2: 菜单还是重复的
**原因**: 服务器未更新  
**解决**: 
```bash
# SSH 登录服务器
cd /home/www/ai.eivie.cn
git pull
```

### 问题 3: 样式不对
**原因**: CSS 文件未加载  
**检查**: 
1. F12 → Network 标签
2. 刷新页面
3. 查找 `sidebar.css`
4. 应该返回 200 状态码

### 问题 4: 折叠后浮动菜单不显示
**原因**: JS 未正确初始化  
**检查**:
1. F12 → Console 标签
2. 查看是否有 JS 错误
3. 确认 `sidebar.js` 已加载

### 问题 5: 移动端抽屉打不开
**原因**: 事件监听未绑定  
**检查**:
1. 确认汉堡菜单按钮存在
2. 检查 Console 是否有错误
3. 尝试重新加载页面

如果出现严重问题，可以快速回滚:

```bash

cd /home/www/ai.eivie.cn

# 查看提交历史
git log --oneline -5

# 回滚到上一个版本
git reset --hard 49ae3a96

# 强制推送（谨慎操作）
# git push -f origin main

# 或者只回滚文件

git checkout 49ae3a96 -- static/index3/css/sidebar.css

git push
```
