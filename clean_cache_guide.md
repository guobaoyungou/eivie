# 封面比例同步修复与缓存清理指南

## 修复已完成

✅ **默认封面比例已更新**：图片生成(`photo_generation`)和视频生成(`video_generation`)组件的默认封面比例已从 `1:1` 改为 `3:4`

✅ **多端同步**：所有平台（微信、支付宝、百度、抖音、QQ、H5、APP）将使用相同的封面比例设置

✅ **现有数据已更新**：页面ID=31的封面比例参数已为 `3:4`

## 多端缓存清理步骤

### 1. 服务器端缓存清理（已完成）
```bash
rm -rf runtime/cache/* runtime/temp/* runtime/log/*
```

### 2. 浏览器端缓存清理（用户操作）
- **微信小程序**：删除小程序重新进入，或在小程序设置中清理缓存
- **支付宝小程序**：删除小程序重新进入
- **百度小程序**：删除小程序重新进入
- **抖音小程序**：删除小程序重新进入
- **QQ小程序**：删除小程序重新进入
- **手机H5**：浏览器设置中清除网站数据
- **手机APP**：重启应用或清除应用数据

### 3. 设计器页面缓存清理
访问设计器页面时按 `Ctrl+F5` 强制刷新页面，清空浏览器缓存。

### 4. CDN缓存（如果有）
如果使用了CDN，需要刷新CDN缓存，确保静态资源更新。

## 验证步骤

1. **设计器页面**：打开设计器，新添加的图片生成/视频生成组件应该默认显示 `3:4` 封面比例
2. **多端预览**：在所有平台预览页面，确认封面比例一致为 `3:4`
3. **保存测试**：修改封面比例设置，保存后查看所有平台是否同步更新

## 注意事项

1. **新组件**：新添加的组件将自动使用 `3:4` 封面比例
2. **现有组件**：已保存的页面会自动继承新的默认值
3. **用户设置**：用户手动设置的封面比例优先于默认值
4. **平台差异**：某些平台可能有自己的缓存机制，需要单独清理

## 如需进一步帮助

如果仍有平台显示旧封面比例：
1. 检查该平台是否有独立的缓存设置
2. 确认该平台的小程序/APP版本是否最新
3. 联系平台技术支持获取缓存清理方法
