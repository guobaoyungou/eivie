# 📱 用户操作指南 - 解决菜单重复问题

## ⚠️ 重要提示

您看到的菜单重复问题是由于**浏览器缓存了旧版本**导致的。我们已经在服务器端完成了修复，现在只需要让您的浏览器重新加载最新版本即可。

---

## ✅ 最简单的解决方法

### 方法一：强制刷新（推荐⭐）

1. 打开网站: `https://ai.eivie.cn/`
2. 按下快捷键:
   - **Windows 用户**: `Ctrl + Shift + R` 或 `Ctrl + F5`
   - **Mac 用户**: `Cmd + Shift + R`
3. 等待页面完全加载
4. 检查左侧菜单是否正常

**这个方法成功率 95%**

---

## 🔧 如果强制刷新无效

### 方法二：清空浏览器缓存

#### Chrome / Edge 浏览器

1. 按 `F12` 打开开发者工具
2. 右键点击地址栏旁边的**刷新按钮**
3. 在弹出菜单中选择"**清空缓存并硬性重新加载**"

或者：
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 勾选"**缓存的图片和文件**"
3. 时间范围选择"**过去 1 小时**"或"**全部时间**"
4. 点击"**清除数据**"
5. 刷新页面

#### Firefox 浏览器

1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 勾选"**缓存**"
3. 时间范围选择"**全部**"
4. 点击"**立即清除**"
5. 刷新页面

#### Safari 浏览器

1. 菜单栏 → **开发** → **清空缓存**
2. 或按 `Cmd + Option + E`
3. 刷新页面

**注意**: 如果看不到"开发"菜单，先去 Safari → 偏好设置 → 高级 → 勾选"在菜单栏中显示'开发'菜单"

---

### 方法三：无痕模式测试

如果您想先测试一下是否修复成功，可以使用无痕模式：

1. 打开浏览器的无痕/隐私模式:
   - **Chrome**: `Ctrl + Shift + N` (Windows) 或 `Cmd + Shift + N` (Mac)
   - **Firefox**: `Ctrl + Shift + P` (Windows) 或 `Cmd + Shift + P` (Mac)
   - **Safari**: `Cmd + Shift + N`
   - **Edge**: `Ctrl + Shift + N`

2. 在无痕窗口中访问: `https://ai.eivie.cn/`

3. 检查菜单是否正常

**如果无痕模式下菜单正常，说明就是缓存问题，清空缓存即可解决**

## ❓ 常见问题

### Q: 为什么会出现这个问题？
**A:** 我们今天对菜单进行了优化升级，但您的浏览器还在使用旧版本的缓存文件。

### Q: 清空缓存会丢失我的数据吗？
**A:** 不会！清空缓存只会删除网页的临时文件，不会影响您的账号、数据或设置。

### Q: 我按了强制刷新，但还是看到旧菜单？
**A:** 请尝试：
1. 完全关闭浏览器
2. 重新打开浏览器
3. 再次访问网站
4. 如果还不行，使用方法二清空缓存

### Q: 其他页面正常，只有首页有问题？
**A:** 在首页按 `Ctrl + Shift + R` 强制刷新首页。

## 💡 小贴士

**为什么需要强制刷新？**

普通刷新（F5）会优先使用缓存，而强制刷新（Ctrl+Shift+R）会强制浏览器从服务器重新下载所有文件，确保您看到的是最新版本。

**多久需要清一次缓存？**

一般不需要主动清理。只有在网站更新后出现显示异常时，才需要清空缓存。
