## 📝 功能使用指南

### Toast提示使用

在JavaScript中调用：

```javascript
// 成功提示
Index3.showToast('操作成功', 'success');

// 错误提示
Index3.showToast('加载失败，请重试', 'error');

// 信息提示
Index3.showToast('正在处理...', 'info');

// 警告提示
Index3.showToast('网络不稳定', 'warning');
```

### 键盘快捷键

- **Tab** - 在可交互元素间移动
- **←/→** - 在Tab按钮间切换
- **Enter/Space** - 激活当前元素
- **Esc** - 关闭模态框

### 数据缓存说明

- 供应商模型数据自动缓存5分钟
- 5分钟内重复访问无需重新加载
- 过期后自动刷新数据

## 🔧 开发者注意事项

### 1. 能力标签数据格式

后端需确保`capability_tags`为数组或JSON字符串：

```php
// 方式1：数组（推荐）
'capability_tags' => ['文生图', '图生图', '高清放大']

// 方式2：JSON字符串
'capability_tags' => '["文生图","图生图","高清放大"]'
```

前端会自动解析。

### 2. CSS版本号更新

如果样式未生效，清除浏览器缓存或更新版本号：

```html
<link rel="stylesheet" href="/static/index3/css/index.css?v=2">
<link rel="stylesheet" href="/static/index3/css/responsive.css?v=2">
```

### 3. JavaScript版本号更新

```html
<script src="/static/index3/js/api.js?v=2"></script>
<script src="/static/index3/js/index.js?v=2"></script>
```

## 🐛 故障排查

### Toast不显示

**原因**：JavaScript未正确加载

**解决**：检查浏览器控制台是否有错误，确认`index.js`已加载

### 骨架屏一直显示

**原因**：API请求失败或超时

**解决**：
1. 检查网络请求状态
2. 查看浏览器控制台错误信息
3. 确认后端接口`/index/model_list_by_provider`正常

### 键盘导航不工作

**原因**：元素未添加tabindex或事件未绑定

**解决**：
1. 检查元素是否有`tabindex="0"`属性
2. 打开浏览器控制台，查看是否有JavaScript错误
3. 确认`initKeyboardNavigation()`已执行

### 能力标签不显示

**原因**：后端数据格式错误

**解决**：
1. 检查后端返回的`capability_tags`字段格式
2. 确保为数组或有效的JSON字符串
3. 查看浏览器控制台是否有JSON解析错误

### 动画卡顿

**原因**：浏览器硬件加速未启用

**解决**：
1. 检查浏览器硬件加速设置
2. 更新显卡驱动
3. 在低端设备上可能需要降低动画复杂度
