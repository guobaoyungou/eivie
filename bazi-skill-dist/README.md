# 八字 + 紫微斗数 双系统命理 Skill

中华传统命理学的可复用 AI Agent skill。融合八字四家投票法（徐乐吾/梁湘润/袁树珊/韦千里）与紫微斗数中州派（陆斌兆+王亭之），双系统交叉验证。

**支持的 AI Agent**（自动检测，多端通用）：
- Claude Code（Anthropic 官方）
- Codex CLI（OpenAI）
- OpenCode（开源 fork）
- 其他兼容 SKILL.md 规范的 Agent

setup 脚本会自动检测已安装的工具并装到对应路径，**装一份所有 Agent 都能用**（通过符号链接共享）。

## 包含内容

- **17 部公共领域典籍**（PD，全文）— 历经数百年验证的元典

- **一键安装脚本**

**包大小：约 60MB**（zip 压缩后）

## 快速开始

**macOS / Linux：**

```bash
unzip bazi-ziwei-mingli-skill-v8.5.zip
cd bazi-skill-dist
bash setup.sh
```

**Windows：**

```cmd
:: 解压 zip 后双击运行
setup.bat
```

或在 PowerShell / cmd 里：
```cmd
cd bazi-skill-dist
setup.bat
```

**WSL / Git Bash 用户**：直接用 `bash setup.sh`，与 macOS/Linux 一致。

setup 脚本会**自动检测**已安装的 AI Agent（Claude Code / Codex CLI / OpenCode），并装到所有对应路径。

按提示选择：
- **Level 1（轻量，零依赖）**：Agent 直接 Read/Grep 查 books/ 原典
- **Level 2（高级，含 LightRAG）**：注入 44M 知识图谱，跨书检索

安装后重启 AI Agent，在对话中直接说：

> 帮我排八字：阳历 1990 年 5 月 15 日 上午 10 点，男，北京

> 戊日生于申月，调候用神是什么？查穷通宝鉴

> 帮我看 2026 年的流年流月，四家方法对照

## 核心特性

### 1. 八字四家投票（ensemble voting）

```
徐乐吾（格局调候） + 梁湘润（三轨+金不换+四规则）
   + 袁树珊（命宫+小限+十六字法） + 韦千里（八步法+转角论）
   ↓
四家一致 = 高置信度
三家一致 = 中置信度（记住少数派警告）
两两分歧 = 低置信度（等事实验证）
```

每家准确率约 70%，四家投票后共识结论显著高于单家。

### 2. 八字-紫微双系统交叉验证

两套独立系统在同一个人身上的结论必须互相印证。八维度交叉校验：财运 / 事业 / 婚姻 / 迁移 / 健康 / 贵人 / 黄金期 / 最差期。

### 3. 流月双轨排法

梁湘润四规则（流月对大运）+ 袁树珊十六字法（流月对大运+小限+命宫）并行，结果对照。

### 4. 双层 RAG 模式

- **模式 A（默认）**：Claude 用 Read/Grep 直接查 books/ 下 17 部 PD 典籍
- **模式 B（可选）**：注入 LightRAG 知识图谱，跨书检索 + 关系推理（详见 `core/rag-usage.md`）

## 铁律（AI Agent 自动遵守）

1. **禁止脚本排盘** — 必须手动推演
2. **禁止反推**（陈素庵） — 不能拿事实凑解释
3. **禁止蒙猜** — 不知道就说不知道
4. **禁止截看大运** — 干支不可割裂
5. **禁止唯一论** — 不能只用一个视角

详见 SKILL.md 第二节。

### 半合规则（最易出错）

半三合**必须带子午卯酉**（四正位）：
- 酉丑 = 半合金 ✓ / 寅午 = 半合火 ✓
- 巳丑 = 不算 ✗ / 寅戌 = 不算 ✗ / 卯未 = 不算 ✗

## 适用场景

- 个人命盘排盘与解读
- 大运/流年/流月决策窗口分析
- 重大人生节点（迁移/婚姻/事业转型）的多角度参考
- 命理学习与典籍研究

## 不适用场景

- 短期赌博/股市单日预测（命理粒度不到日内）
- 替代医疗/法律/金融专业意见
- 宿命论决策
- 对他人做隐私评判

详见 ETHICS.md。
