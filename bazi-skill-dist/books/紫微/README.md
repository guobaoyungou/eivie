# 紫微斗数书籍

## 紫微斗数全书 — 佚名（正统道藏版，明）

斗数经典原典，三卷。来源：ctext.org（中国哲学书电子化计划）。

**核心内容：**
- 卷一：论次序（定时刻→起八字→立格局→排星辰→立坐命→起大运→起大限→书化曜→书喜忌→排吉凶）、十二时辰分宫论命、各时刻先克父母
- 卷二：星曜性质、格局论断
- 卷三：诸星入宫详论

**速查：**
- 排盘步骤 → 卷一开篇"论次序"十步法
- 时辰分宫 → 卷一（子时天库宫、丑时红鸾宫...）
- 星曜格局 → 卷二、卷三

## 紫微斗数讲义 — 陆斌兆（民国）

中州派代表作，星曜性质按十二宫逐一详论。

**核心内容：**
- 紫微斗数概述：与天文学关系、与子平八字对比、推算方法、闰年处理
- 天地人三盘图
- 14主星逐星详论：紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军
- 每星分12宫论述：命宫、兄弟宫、妻宫、子女宫、财帛宫、疾病宫、迁移宫、交友宫、事业宫、田宅宫、福德宫、相貌宫

**速查：**
- 某星在某宫表现 → 按星名章节 → 对应宫位小节
- 斗数与八字区别 → 概述《紫微斗数和子平八字》
- 排盘方法 → 概述《推算紫微斗数的方法》

## 紫微斗数精成 — 大德山人（当代）

系统教材，上下两编。查阅式风格，排盘后按宫位星曜对号入座。

**核心内容：**
- 上编：基础知识（阴阳五行、干支、纳音）、排盘方法步骤、术语解释、星曜分级分类、十二宫意义、分析推演（格局、事业、财运、疾厄、六亲、婚姻）、女命论、50类星曜逐一落命宫参断
- 下编：后天运势、大小限分析、四化分析论断、梅花易数式灵活预测

**速查：**
- 排盘步骤 → 第二章
- 某星曜性质 → 第八章（50类星曜逐一详论，含主星+辅星+杂曜）
- 格局论断 → 第六章第四节
- 婚姻预测 → 第六章第九节
- 四化分析 → 下编

## 命理天机 — 周德元（当代）

规则运用与分析，流派梳理最详细的一本。团结出版社2013年。

**核心内容：**
- 丛书总序：神秘文化的辨析立场（不迷信、不棒杀）
- 前言：命理学史（孔孟命观→李虚中→徐子平→陈抟紫微）
- 紫微斗数流派详细梳理：
  - 三合派：中州派、紫云派、现代派、天机派、沈氏紫微、占验派、恭鉴老人派
  - 四化派：河洛派、钦天派、仙宗派
  - 其他：一叶知秋派、科技派
- 第一章：命盘（十二宫与八卦对应、五行局、命主身主）
- 第二章：十二宫详论
- 第三章：星曜（分类、五行属性、财富星、桃花星）

**速查：**
- 紫微斗数各流派特点 → 前言（最全的流派对比资料）
- 斗数与子平术区别 → 前言
- 四化星概念 → 前言（飞星特性）
- 星曜分类速查 → 第三章
- 十二宫含义 → 第二章
