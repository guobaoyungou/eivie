# 八字（子平术）书籍

## 渊海子平 — 徐大升（宋）

八字命理体系奠基之作，确立"日干为主"的论命方法。

**核心内容：**
- 五行生克制化总则（金旺得火方成器皿...）
- 十干十二支暗藏、地支藏遁
- 论日为主：年为根、月为苗、日为花、时为果
- 论月令提纲、太岁吉凶、征太岁
- 论大运（天干五运、地支六气）、转角接木
- 论疾病（五行对应脏腑肢体）

**速查：**
- 五行生克口诀 → 《论五行生克制化》
- 天干地支藏遁 → 《地支藏遁歌》
- 日干见各干关系 → 开篇"以甲为例"
- 大运吉凶 → 《论大运》
- 疾病对应 → 《论疾病》

## 子平真诠 — 沈孝瞻（清）

格局用神理论集大成，共 48 章，体系最严整。

**核心内容：**
- 卷1-7：阴阳干支基础理论
- 卷8：论用神（"八字用神，专求月令"）
- 卷9：论用神成败救应
- 卷10-14：用神变化、纯杂、高低、配气候
- 卷15：相神紧要
- 卷16-30：杂气、墓库、四吉神、四凶神、外格等
- 卷31-46：逐一论正官、财、印、食神、偏官、伤官、阳刃、建禄月劫及取运
- 卷47：杂格

**速查：**
- 用神怎么取 → 卷八《论用神》
- 格局成败 → 卷九（成败救应，极详细）
- 某格局取运 → 卷32/34/36/38/40/42/44/46
- 调候 → 卷十四《论用神配气候得失》
- 合化规则 → 卷四《论十干配合性情》、卷五《论十干合而不合》

## 三命通会 — 万民英（明）

命理百科全书，12卷，涵盖纳音、格局、神煞、干支全部理论。

**核心内容（简体版）：**
- 卷一：五行生成、生克、支干源流、纳音总论、60甲子性质
- 卷二：论天干、地支、纳音取象
- 卷三：论格局（正官、偏官、印、财、食神、伤官等）
- 卷四-六：论六亲、论疾病、论行运
- 卷七-八：论女命、论小儿
- 卷九-十二：诸格局详论、命例

**速查：**
- 纳音五行取象 → 卷一《论纳音取象》
- 60甲子逐个性质吉凶 → 卷一《释六十甲子性质吉凶》
- 某格局详论 → 卷三起
- 神煞 → 卷二

## 滴天髓阐微 — 京图原著/刘基注/任铁樵阐微（宋原/明注/清阐微）

五行生克哲学巅峰，任铁樵大量实战命例佐证。

**核心内容：**
- 通神论34章：天道、地道、人道、知命、理气、配合、天干、地支、形象、方局、八格、体用、精神、月令、衰旺、中和、源流、通关、官杀、伤官、清浊、真假、刚柔、顺逆、寒暖、燥湿、隐显、众寡等
- 六亲论29章：夫妻、子女、父母、兄弟、何知章、女命、小儿、才德、性情、疾病、岁运等

**速查：**
- 五行旺弱判断 → 《衰旺》《中和》
- 格局高低 → 《八格》《体用》《精神》
- 调候 → 《寒暖》《燥湿》
- 用神通关 → 《通关》《源流》
- 何知章（断命口诀） → 六亲论《何知章》
- 女命 → 六亲论《女命章》

## 李虚中命书 — 李虚中（唐）

最早的命理专著，60甲子纳音论命体系，以年为主。

**核心内容：**
- 卷上：60甲子纳音逐一论性质（甲子天官藏...乙丑禄官承...）、贵神优劣、贵合贵食
- 卷中：通理物化（天地阴阳哲学）、真假邪正、升降清浊
- 卷下：衰旺取时、三元九限、天承地禄

**速查：**
- 某甲子纳音性质 → 卷上逐条查
- 贵人规则 → 卷上《论贵神优劣》
- 五行哲学基础 → 卷中《通理物化》

## 命理约言 — 陈素庵著/韦千里选辑（清/民国）

精炼实用，法+赋+论三位一体。原书10卷，精选4卷。

**核心内容：**
- 卷一（法48篇）：看命总法、格局法、用神法、月令法、正官法、偏官法、食伤法、从化格、暗冲暗合、六亲法、贵贱贫富寿夭、女命法等
- 卷二（赋20篇）：总纲赋、格局赋、行运赋、流年赋、十神各赋、从化赋、暗冲暗合赋、女命赋
- 卷三（论48篇）：天干论、地支论、合冲刑害论、神煞论、纳音论等
- 卷四：杂论24则 + 张神峰辟谬论11则

**速查：**
- 看命步骤 → 卷一《看命总法一/二》
- 格局分类速查 → 卷一《看格局法》（列出所有正变格局）
- 用神取法 → 卷一《看用神法》
- 某十神详论 → 卷二对应赋
- 神煞真假 → 卷三《诸神煞论》

## 穷通宝鉴 — 余春台编（清）

调候用神专著，按月令逐一论十天干宜忌。

**核心内容：**
- 十天干分十二月论用神宜忌
- 以五行寒暖燥湿为核心
- 春木、夏木、秋木、冬木各有不同取用

**速查：**
- 某日干某月用什么 → 直接按"X日干+Y月"查找
- 调候优先级 → 冬用火暖、夏用水润为基本原则

**注意：** azw3 格式，需 Kindle 或 Calibre 阅读。
