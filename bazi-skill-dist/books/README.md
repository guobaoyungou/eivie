共 **17 部**公共领域典籍，全部为作者去世满 50 年的著作，可自由使用、复制、分发。

当代受版权保护的著作（梁湘润、韦千里、王亭之、邵伟华、徐伟刚等）**未在本目录分发原文**，其方法论已提炼到 `../core/` 下的通读笔记中（属于学术研究的合理使用范围）。

如需当代著作原书，请购买正版支持作者。详见根目录 `../README.md` 第三节。
