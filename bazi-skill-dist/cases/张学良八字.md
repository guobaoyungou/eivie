空亡: 甲辰旬空寅卯 = 食伤(木)落空亡，原局本就无木 = 空上加空

## 核心特征

### 1. 天干全金水 = 有力无方向
聪明绝顶+资源极丰，但没有输出通道。一辈子不知道自己想干什么。

### 2. 双羊刃(日子+时子) = 刚烈冲动
西安事变 = 双羊刃冲动的极致表现。冲动完了不知道下一步 = 没有食伤。

### 3. 子丑合土 = 根基变枷锁
壬水帝旺根(子)被年支丑合化为官杀(土) = 成也萧何败也萧何。

### 4. 丙辛暗合 = 正印被偏财合走
学识被女色分散。少帅风流 = 丙辛暗合的表现。

### 5. 天德在偏印(庚) = 保护方式是"关起来"
软禁54年反而保了他命。双羊刃+无食伤如果自由 = 大概率早死。
