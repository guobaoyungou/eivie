# 毛泽东八字命盘（卯时盘）

## 基本信息
- 阳历: 1893年12月26日
- 农历: 癸巳年十一月十九日
- 出生地: 湖南韶山（经度约112.5E）
- 真太阳时修正: 北京时间约7:15（日出） → 真太阳时约6:44 → 卯时
- 性别: 男
- 四柱: **癸巳 甲子 丁酉 癸卯**

## 时辰争议
- 传统排法: 辰时（甲辰） → 双印护身，温和大贵
- 真太阳时: 卯时（癸卯） → 双杀夹身，极端强人
- 本文采用卯时，因事实验证更准确（详见验证对照章节）

## 格局

月令子水，子中本气癸水(七杀)。癸水透在年干+时干 → **七杀格**。

甲木正印透月干化杀 → **杀印相生格**

但: 双杀(癸x2) vs 单印(甲x1) = 杀力过重，印制化不足 = "成中有破"

用神排序:
1. 印星(木) — 化杀生身，最急需
2. 比劫(火) — 帮身
3. 食神(己土) — 制杀

忌神:
- 财星(金) — 坏印，金克木则印无力化杀
- 官杀(水) — 再加杀更凶

## 核心特征分析

### 1. 双七杀夹身 — 最核心的结构

癸(年) + 丁(日主) + 癸(时) = 七杀从两头夹攻

七杀=权力/暴力/压迫/革命/军事。双杀夹身的人:
- 一辈子在权力漩涡中
- 要么被权力碾碎，要么驾驭权力碾碎别人
- 没有中间状态

甲木正印在月干（中间位置）挡了一下 = 有思想/理论武装。但一印对双杀 = 印挡不住，杀力外溢。

### 2. 子卯刑 = 无礼之刑

"无礼"不是不懂礼貌，是不遵守既有秩序。
- 子=七杀(权力)，卯=偏印(非正统思想)
- 七杀和偏印相刑 = 用非正统的方式夺取权力
- 对应: 农民革命、游击战、农村包围城市，不走科举/选举的正途

### 3. 卯酉冲 = 冲配偶宫

- 四任妻子（杨开慧被杀、贺子珍出走、江青）
- 子女有灾（毛岸英朝鲜战死、多个子女失散/夭折）
- 时支冲日支 = 子女宫冲配偶宫 = 配偶和子女互相牵连

### 4. 巳落空亡 = 根基虚，火空则明

- 出身农民（根基虚=不是世家贵族）
- "火空则明" = 越暗的环境丁火越亮 = 照亮天下
- 空中楼阁 = 建立的政权根基不实（大跃进/文革的虚妄）
- "星星之火，可以燎原" — 就是这个盘的写照

### 5. 天德月德不入命 = 无保护

没有化凶机制。一旦走忌运，凶性全部落实。大起大落，无人可挡。

## 大运详析

### 第1步: 癸亥 (6-15岁, 1899-1908) — 忌运

- 癸=七杀，加上原局双七杀 = **三杀齐攻**
- 亥中壬(正官)+甲(正印) = 官杀混杂
- **亥冲巳!** 冲掉丁火唯一的根!
- 少年极苦，父亲严厉
- 实际: 父亲毛顺生是严父，逼他种田做生意，父子冲突激烈

### 第2步: 壬戌 (16-25岁, 1909-1918) — 混合

- 壬=正官，官杀混杂继续
- 戌中戊(伤官)+辛(偏财)+丁(比肩)
- **伤官见官!** 壬(正官)遇戊(伤官) = 不服管、反体制
- 戌中丁(比肩)帮身 = 终于有人帮了
- 丁壬合木 = 正官合身 = 有人赏识/提拔
- 实际: 离开韶山求学(1910)，湖南一师(1913-1918)，遇到杨昌济(恩师=正印)
- 伤官见官 = 反封建、反旧制度 = 五四运动前奏

### 第3步: 辛酉 (26-35岁, 1919-1928) — 关键运!

- 辛=偏财，酉=偏财根
- 大运酉+原局酉 = **酉酉并临!** 偏财力极强
- 卯酉冲在大运中被加倍
- 偏财 = 机遇、偏门收获
- 实际: 建党(1921)、国共合作、农民运动、秋收起义(1927)、上井冈山(1928)
- 偏财运 = 不走正途，走偏门起家

### 第4步: 庚申 (36-45岁, 1929-1938) — 险中求变

- 庚=正财，申中庚壬戊 = 结构复杂
- **巳申合!** 大运申+年支巳 = 巳火根被合走!
- 申中壬水(正官) = 官星暗中助力
- 大运中伤官(戊)制杀(癸) = 用谋略翻盘
- 实际: 长征(1934-1935)、遵义会议(1935)重新掌权、延安时期
- 巳申合 = 根被合走 = 被迫离开根据地(长征)
- 遵义会议 = 伤官制杀 = 用谋略翻盘

### 第5步: 己未 (46-55岁, 1939-1948) — 最关键的好运!

- 己=**食神!** 食神制杀!
- 未中己(食神)+丁(比肩)+乙(偏印) = 食神+比肩+偏印三位一体
- 食神制双七杀 → 杀被制住 = **权力到手!**
- 比肩丁火帮身 = 终于有强力帮扶
- 偏印乙木 = 思想武装(毛泽东思想成体系)
- 实际: 整风运动(1942)、抗战胜利(1945)、解放战争(1946-1949)、建国(1949年10月)
- **建国在这步运的最后一年! 食神制杀大功告成!**

### 第6步: 戊午 (56-65岁, 1949-1958) — 巅峰但埋祸

- 戊=**伤官** 透干 = 才华外放、锋芒毕露、目空一切
- 午=丁火帝旺 = 日主最旺的位置，身终于强了
- 午中丁(比肩)帮身 + 己(食神)继续制杀
- 但伤官的本质是"泄"不是"制" = 消耗自身去表现
- 实际: 建国初期、抗美援朝、三大改造、**大跃进(1958)**
- 伤官 = 好大喜功、追求表现 = "赶英超美"
- 毛岸英战死(1950) = 伤官运子女易有灾
- 1958戊戌: 伤官伏吟(大运戊+流年戊) = 好大喜功到极点

### 第7步: 丁巳 (66-75岁, 1959-1968) — 劫难运

- 丁=比肩，巳中丙(劫财)+庚(正财)+戊(伤官)
- **比肩+劫财 = 争权!** 身边的人跟他争
- 巳到位 = 巳火根基回来，引动原局巳的空亡
- 比劫争权 = 权力斗争白热化
- 实际: 庐山会议(1959)、大饥荒(1959-1961)、**文化大革命(1966)**
- 比肩争权 = 彭德怀vs毛、刘少奇vs毛
- 文革发动 = 劫财运的极端表现 = 用暴力清除异己

### 第8步: 丙辰 (76-85岁, 1969-1978) — 终局

- 丙=劫财，辰中戊(伤官)+乙(偏印)+癸(七杀)
- 劫财继续 = 权力斗争不止
- 辰中癸(七杀) = 七杀又从地支冒出来
- **辰酉合** = 大运辰+日支酉 = 合配偶宫 = 江青得势
- 1976年流年丙辰 = **大运丙辰+流年丙辰 = 伏吟!**
- "反吟伏吟，哭泣淋淋" = 去世
- 实际: 九大(1969)、林彪事件(1971)、四人帮、毛去世(1976.9.9)

## 格局总定性

### 层次: 上等偏上（极端型）

七杀格+杀印相生，但双杀夹身+单印制杀不足 = "成中有破"。
不是温和的大贵，是**险中求贵、以命搏天**的极端格局。

### 核心结构

**双杀夹一丁，单印挡不住。子卯刑无礼，卯酉冲妻儿。巳空根基虚，火空照天下。食神运制杀，天下归手中。**

### 一生轨迹的命理逻辑

1. 身极弱+双杀夹身 = 出身底层，一辈子被权力裹挟
2. 子卯刑(无礼之刑) = 不走正途，用非正统方式夺权
3. 巳空+火空则明 = 没有根基反而照亮天下，星星之火可以燎原
4. 己未运食神制杀 = 46-55岁是命中注定的成功窗口
5. 伤官运(戊午)+比劫运(丁巳丙辰) = 成功后开始消耗、内斗、极端化
6. 天德月德不入命 = 无化凶机制，大起大落毫无缓冲
7. 丙辰伏吟 = 终局

### 丁火的本质

丁火是星火、蜡烛之火。不像丙火是太阳需要大地承载，丁火是**越暗越亮**。冬天子月、身极弱、根基虚空 = 最暗的环境里，丁火的一点光反而照得最远。

这就是为什么身极弱反而成就最大——丁火不需要强，需要的是暗。天下越乱(七杀越旺)，他越有用武之地。

## 此盘的方向定性

**毛泽东这个盘是用来打天下的，不是用来过日子的。**

- 日主丁火身极弱：易折但能借势
- 格局七杀格杀印相生 = 贵格（不是富格）
- 核心链条癸→甲→丁（杀化权）：靠对手喂养自己
- 卯酉冲 = 冲配偶宫，子卯刑 = 无礼之刑：婚姻一生动荡
- 巳空 = 唯一根基虚：差点被灭无数次
- 七杀在时柱 = 子女有灾
- 上限：帝王。下限：极低。波动性：极大。

帝王格的代价就是不能"过日子"——下限风险极大，正常生活的稳定性几乎没有。
若要与"富格（食神生偏财）"或"贵格（财官印三全）"对比，可参见 cases/邵逸夫八字.md 与 cases/张学良八字.md。
