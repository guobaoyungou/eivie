## 核心特征分析

### 1. 天干四位一体链条

辛(印) → 壬(身) → 甲(食) → 丁(财)

印生身 → 身生食 → 食生财 = 一条链从头到尾不断不破，每一环都透干。
四环全透 = 最完整。两环（食→财）短但有力，但层次低于四环。

### 2. 丁壬合木 = 财富飞轮

做电影(甲木食神=文化/艺术) → 电影赚钱(丁火正财) → 钱再投入做电影 = 永动机。
合化方向是食伤(喜神) = 越赚越有才，越有才越赚。

### 3. 地支无冲无刑无害 = 长寿107岁

对比毛泽东（卯酉冲+子卯刑 = 一辈子风暴）和一般有寅申冲的盘（多波动），邵逸夫地支全安静。

### 4. 华盖+食神 = 文化商人

华盖主艺术，食神主表达创造，两者结合 = 天生做文化产业的命。

- 己=正官，有管束

- 实际: 跟随兄长学做电影，从底层做起

### 第3步: 戊申 (24-33岁, 1931-1940) — 混合偏好
- 戊=七杀，竞争激烈
- 身强能担杀 = 开疆拓土
- 实际: 在南洋(新加坡马来亚)开拓电影市场

### 第4步: 丁未 (34-43岁, 1941-1950) — 好运!
- 丁=正财，跟年干丁并临 = 正财力量翻倍
- 大运丁+日主壬 = 丁壬合木 = 财来化才

### 第5步: 丙午 (44-53岁, 1951-1960) — 好运!

- 偏财+正财齐旺 = 开始真正发大财
- 实际: 1958年成立邵氏兄弟(香港)，大举投资电影

### 第6步: 乙巳 (54-63岁, 1961-1970) — 好运!

## 格局总定性

### 层次: 上等（稳健型大富贵）

食神生正财格，天干四环链条完美(印→身→食→财)，丁壬合木化喜，
地支无冲无刑无害，天月二德在食神(用神)，华盖在年支。

### 核心优势

1. 天干链条四环一气贯通，每一步都透干落实
2. 丁壬合木化喜 = 财富良性循环飞轮
3. 地支无冲无刑无害 = 一生平稳，长寿基础
4. 天月二德保护用神(食神) = 才华不会被废
5. 34-73岁连续40年好运 = 黄金40年
6. 五行偏均衡 = 健康长寿

### 命格定位

邵逸夫的盘是"天生做文化生意的命"（食神生正财+丁壬合木+地支安稳）。

这是顺水行舟型富格——区别于"搏出来的富"（食神生偏财+寅申冲带动力的盘）。
都能成富，但顺水行舟舒服得多。

### 为什么是顶级稳健富格

1. 天干四环全透（印→身→食→财）= 一条链不断
2. 丁壬合木化喜 = 财来往喜神方向走
3. 地支无冲无刑无害 = 一生稳定
4. 天月二德落在用神（食神时干） = 用神不会被废
5. 好运 40 年（34-73岁） = 黄金期超长
