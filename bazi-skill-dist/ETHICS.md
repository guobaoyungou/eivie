# 伦理与免责声明

## 命理学的定位

命理学是**概率性倾向描述**，不是**宿命决定论**。

四柱八字与紫微斗数描述的是一个人在特定时空下的能量结构与可能轨迹。它的预测能力来自于千年实证的统计经验，而不是神谕。

> 命好不努力一样穷，命差不放弃也能活。
>
> 知命非为认命，知命是为正命。

## 本 skill 的定位

本 skill 是**研究与自我认知工具**，不是：

- 不是医疗诊断工具（命理不能替代医生）
- 不是法律咨询（不能替代律师）
- 不是金融投顾（不能替代专业理财师）
- 不是心理治疗（命理咨询 ≠ 心理治疗）
- 不是赌博工具（命理不到日内交易粒度）

## 使用边界

### 适合做什么

- 帮助理解自己的性格倾向、能量结构
- 对人生重大节点（搬迁、婚姻、转型）提供参考视角
- 学习中华传统文化的认识论与方法论
- 作为多角度决策框架中的一个维度

### 不适合做什么

- **不要用于做出医疗决定**：身体不舒服请去看医生
- **不要用于法律纠纷**：请咨询律师
- **不要用于全仓押注金融决策**：命理是一个维度，不是全部
- **不要拿来批判他人**："你命差所以不行"是错误的
- **不要用于歧视**：性别、出生时辰不应成为人际偏见的依据
- **不要用于操纵他人**：替别人算命要尊重对方意愿

## 关键认识

### 1. 命理是后见之明的总结，不是先验真理

千年命理实践积累的是**模式识别**。与现代统计学一样，模式识别有概率，没有 100%。

### 2. 同一个八字的两个人命运可以差很多

袁树珊命谱多次论证：同造不同地、不同家世、不同时代，命运悬殊。**八字描述潜力空间，环境与选择决定具体路径。**

### 3. 命理预测有锚点验证机制

陈素庵铁律：**"非稽其已往，无以测其将来。"**

任何命理判断必须先用已知事实验证框架。验证不通的，要反思框架，不要硬解释。

### 4. 命理不能替代努力

命理告诉你：什么时候顺，什么时候逆，什么方向适合你，什么方向不适合。

但即使最好的运势，你不动也什么都不会发生。即使最差的运势，你保持理性也能减少损失。

## 关于隐私

本 skill 处理的是**生辰八字（出生年月日时）+ 出生地**。

这些信息属于个人隐私。使用建议：

- 不要随便把别人的生辰八字交给他人解读
- 不要把命盘详情发到公开平台
- 替他人排盘前，明确征得对方同意
- 命盘信息保存在本地，不要上传云端

## 反对的使用方式

本 skill 明确反对以下使用方式：

1. **用命理做歧视**：种族、性别、出身、命格高低不应成为社会评价依据
2. **用命理勒索情感**：用"你命中注定要嫁给我"等话语操纵他人
3. **用命理逃避责任**："命中注定我会犯这个错"是错误归因
4. **用命理做大额收费**：传统命理咨询是讨论，不应成为金融诈骗包装
5. **用命理替代医疗**：命理看到健康倾向就提醒注意，但绝不能替代医疗诊断
