# 八字 + 紫微斗数 双系统命理 Skill

## 一、本 skill 是什么

基于中华传统命理学经典文献的排盘与论命系统。通读 30+ 部典籍后提炼出的实务方法论（其中 17 部公共领域典籍随包提供全文，当代著作的方法论以通读笔记形式呈现）：

- **八字四家投票法**：徐乐吾（格局调候）+ 梁湘润（三轨+金不换+四条规则）+ 袁树珊（十六字法）+ 韦千里（八步法）四家并行，投票定共识
- **紫微斗数中州派**：陆斌兆讲义 + 王亭之注释，原典征验
- **双系统交叉验证**：八字-紫微在财运/事业/婚姻/迁移/健康/贵人/黄金期/最差期八维度上互相校验

## 二、最重要的铁律

### 铁律 1：禁止脚本排盘

所有排盘必须手动推演。脚本排盘曾出现的严重错误：
- 巳丑误判半合金（实际不带四正位，不算半合）
- 寅戌误判半合火（同样不算）
- 漏判命宫天干合
- 流月评级两级反转

**手动排盘逐步推理，每一步可检查，不会出现系统性偏差。**

唯一允许计算器辅助的：日柱查万年历（纯数学计算）、节气日期查询。

### 铁律 2：禁止反推（陈素庵）

> "非稽其已往，无以测其将来。"

必须先用已知事实验证框架，验证不通的不能硬解释。**禁止拿事实去凑八字。**

### 铁律 3：禁止蒙猜

不知道就说不知道。命理学里"可能""大概""应该是"都是禁词，要么有典籍出处，要么有锚点验证，要么承认不知道。

### 铁律 4：禁止截看大运

任铁樵：一运看十年，切勿上下截看。陈素庵反对"上五年看天干，下五年看地支"。**先通观十年整体性质，再"着重"前后五年，不是"只看"前五或后五。**

### 铁律 5：禁止唯一论

不能只用一个视角断命：
- 唯格局论：沈孝瞻"木逢冬水，虽透官星，亦难准贵"
- 唯神煞论：张神峰"此只把一字以定生死，实谬说也"
- 唯用神论：陈素庵"体不成则用无所附"
- 唯身强弱论：梁湘润"日主强弱是入门之优先，不是子平法全部涵义的第一优先"

## 三、八字十五步排盘流程

详见 `core/methodology.md`，要点：

1. **编码**：阳历 → 真太阳时（经度差+均时差）→ 四柱+藏干
2. **识日主**：滴天髓十干论，区分阳干阴干
3. **观月令**：月支本气是否透干？季月（辰未戌丑）注意分日用事
4. **日主强弱**：旺看月令，强弱看党众。注意"衰而强"和"旺而弱"
5. **调候**：穷通宝鉴十干配十二月。调候为急时压过格局
6. **格局**：取格优先级 = 月支本气透 > 中余气透 > 月干坐根 > 三合三会
7. **形象**：清浊真假，源流通关
8. **用神**：格局/通关/病药/专旺/调候五种，不可混用
9. **性情**
10. **疾病**
11. **六亲**：宫位+十神参合，"妻以配身，子以为后"
12. **财官**：何知章富贵贫贱寿夭判断纲领
13. **大运流年**：五道筛子（生旺库凶年/天罗地网/伏吟反吟/日犯岁君/格局喜忌）
14. **神煞辅助**：陈素庵"真者精择而存之，妄者悉举而削之"
15. **综合判断**：必须验证已知事实

补充步骤：A 命宫（袁树珊）/ B 小限 / C 空亡 / D 大运标注 / E 调候与地理 / F 身强弱修正 / G 病药说

## 四、八字四家投票方法

四家并行 = ensemble 投票法（与机器学习随机森林同构）：

| 家 | 核心方法 | 强项 |
|----|---------|------|
| 徐乐吾 | 格局+调候 | 体系完整，子平真诠注释权威 |
| 梁湘润 | 三轨+金不换+四条规则 | 实务派，流月规则最精 |
| 袁树珊 | 命宫+小限+十六字法 | 看见别人看不见的暗损 |
| 韦千里 | 八步法+转角论 | 大运转角点定位精准 |

**置信度判定：**
- 四家一致 → 高置信度
- 三家一致 → 中置信度（记住少数派警告）
- 两两分歧 → 低置信度（等事实验证）

每家准确率约 70%，四家投票后共识结论准确率显著高于单家。

## 五、半合规则（最易出错）

梁湘润明确规定：**半三合必须带子午卯酉（四正位）**，否则不算。

- 酉丑 = 半合金 ✓（带酉）
- 巳酉 = 半合金 ✓（带酉）
- 寅午 = 半合火 ✓（带午）
- **巳丑 = 不算半合 ✗**（不带四正位）
- **寅戌 = 不算半合 ✗**（不带四正位）
- **卯未 = 不算半合 ✗**（不带四正位）

脚本排盘最常错的就是这条。

## 六、流月批导（必须两套并行）

### 梁湘润四条规则（流月只看与大运的关系）

1. **月运同甲子**（流月=大运伏吟）→ 大运效应集中爆发
2. **月运反吟**（流月冲大运）→ 动荡
3. **月运双合**（天干地支都合大运）→ 变质
4. **月运拱合**四柱+大运

流月与四柱无刑冲合会关系。流月与流年无刑冲合会关系。

### 袁树珊十六字法（流月看大运+小限+命宫）

小限推算：1岁小限 = 命宫干支，逐年逆行。

十六字 = 四柱 8 字 + 命宫 2 字 + 小限 2 字 + 大运 2 字 + 流年 2 字。

流月逐月对照大运（梁四条）+ 小限（冲合害伏吟）+ 命宫（冲合害伏吟），三层叠加。

### 两套结果对照

**两套结果必须对照，不一致时标注差异原因。**

## 七、紫微斗数核心方法

### 7.1 排盘要素

- **十二宫**：命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、仆役、官禄、田宅、福德、父母
- **十四主星**：紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军
- **辅星煞星**：左辅、右弼、文昌、文曲、天魁、天钺、禄存、擎羊、陀罗、火星、铃星、地空、地劫
- **四化**：化禄、化权、化科、化忌

### 7.2 命宫与身宫

- **命宫**：先天格局（出生时定，看本质）
- **身宫**：后天追求（看人生取向，必落于命/夫/财/迁/官/福六宫之一）

### 7.3 三方四正

任一宫的判断需看其三方四正：本宫 + 对宫（迁移/夫妻/福德等）+ 三合方两宫。

### 7.4 紫微大运流年法

- **大运（十年）**：从命宫起，按五行局起运岁数顺/逆排
- **流年**：从大运宫起子年，顺数到流年地支
- **流年四化**：以流年天干起四化，飞入原盘十二宫，看落到哪宫激活什么

### 7.5 关键格局口诀（陆斌兆讲义+紫微斗数全书）

- **紫微七杀化权，反作祯祥**：紫七同宫见化权 = 帝星压杀
- **武贪庙旺，先贫后富**：武曲贪狼在丑未，三十后发福
- **限到亥卯未，骤发骤富**：三方四正在亥卯未的人，逢亥卯未运/年发福
- **天府临迁移，出外得福**：天府最佳位置之一
- **武曲化忌+空劫照财帛**：财来财去，结构性漏洞，需配偶掌财

### 7.6 紫微化忌的化解机制

化忌不是绝对凶。讲义提供五重化解路径：
1. 禄存照同宫 → 财路不断
2. 主星庙旺 → 跌倒能起
3. 化禄从三方来 → 对冲补血
4. 迁移宫吉 → 外部保护
5. 配偶宫吉 → 人事补救

操作原则：**赚到就走，不恋战；多线收入；异地理财。**

### 模式 B：LightRAG 图谱检索（推荐用于深度研究）

`lightrag/merged.json`（44MB）是预构建的命理学知识图谱：**45895 实体 + 108184 关系**，从 36 部典籍中通过多轮 LLM 抽取得到（含已删除原文的当代著作的方法论摘录）。

**何时用 RAG：**
- 通读笔记 + books/ Grep 都查不到的问题
- 跨书对比、图谱推理（"X 和 Y 之间有什么关系"）
- 模糊查询（不知道精确关键词，只有大致主题）

**何时不用：**
- 简单排盘步骤（按 methodology.md 走）
- 已知精确典籍位置（直接 Read/Grep 更快）
- 纯计算问题

**调用方式（注入 knowledge-mcp 后）：**

```
mcp__knowledge_mcp__retrieve(
    kb_name="bazi-mingli",
    query="武曲化忌在财帛宫的化解方法，各家说法对比",
    mode="hybrid",  # naive/local/global/hybrid，默认 hybrid
    top_k=10
)
```

**铁律**：不要只读 RAG 概括，**必须看返回的原文片段并交叉验证**。这违反"非稽其已往"铁律。

详细安装步骤、四种查询模式选择、Step-by-step 流程、实战示例、回退方案，**全部见** `core/rag-usage.md`。

### 选择建议

| 使用场景 | 推荐模式 |
|---------|---------|
| 偶尔排盘 / 学习入门 | 模式 A 即可 |
