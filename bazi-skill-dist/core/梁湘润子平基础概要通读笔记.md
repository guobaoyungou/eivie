## 一、梁湘润的核心方法论：层次观

梁湘润的最大贡献不是某个具体技法，而是他明确提出了**子平法的推理层次**，并且反复警告不要把入门层次当成全部。

### 1.1 三阶段九个月

- 第一阶段（3个月）：入门基础——背熟刑冲合会、十神生克、六亲所属、十干禄绝
- 第二阶段（3个月）：格局与调候的兼顾达悟
- 第三阶段（3个月）：诸家比判融合 + 全流神煞会合

### 1.2 "日主强弱"不是论命第一优先

这是梁湘润最重要的一句话：

> "日主强弱是入门之优先，不是子平法全部涵义的第一优先。如果在初阶段之时，先被日主强弱视为论命程式上的第一优先，日后将终身困于扶抑日主的小范围之中矣。"

他举了一个极端例子：丙日主生酉月，日主不得令，但仍然以壬水为第一优先——不论日主如何弱，不论入任何格，缺水则全盘皆堕。

**结论：调候 > 日主强弱。** 日主强弱的范围，在调候用神之后。

### 1.3 用神的五个层面

梁湘润把"用神"这个被滥用的词明确分为五种，这比徐乐吾清晰得多：

| 用神类型 | 含义 | 适用场景 |
|----------|------|----------|
| 格局用神 | 顺用、逆用的用神 | 格局成败的主线 |
| 通关用神 | 破格时的调节用神 | 格局受损时的补救 |
| 病药用神 | 调候忌神的对治 | 食神制杀、伤官配印等 |
| 专旺用神 | 从旺、从势格中的用神 | 特别格 |
| 调候用神 | 十干对十二月支的用神 | 寒暖燥湿的根本需求 |

### 1.4 十神定位法：独立于格局的速判系统

梁湘润独创的实战方法——不依赖格局判断、不依赖调候，单纯用十神在四柱的位置+坐支+神煞来推论吉凶。

核心原则：
- 十神定位可以与格局并论，也可以与调候并论，也可以独立使用
- 初学者用此法"在二个月内可对八字勾画50%以上的吉凶范围"
- 然后再进一步研究格局的顺逆用、调候等主流层面

精华规则举例：
- 比肩坐空亡：在年不利父，在月不利兄弟
- 偏印天干二见或三见：主迟婚、变缘、独身
- 偏印坐月支专位入格：以艺文技专长为业
- 正官坐桃花：主夫良（唯一桃花论吉的情况）
- 正财坐正官同根透出：书香门第
- 日支专位坐正财（戊子日）：得勤俭持家的贤淑妻室
- 壬午日、癸巳日：自坐财官印，为"禄马同乡"，大吉
- 七殺坐桃花遇刑冲：因感情而引祸
- 食神坐羊刃：劳碌之命
- 傷官坐羊刃：背禄逐马，主十五年力不从心

## 二、顺用逆用体系

梁湘润把沈孝瞻的理论组织成了极清晰的框架。

### 2.1 顺用四格

正官、正印、食神、正财——只能泄不能克。

"格局强而日主弱"时，不可反克格局，只能用泄的方式调停。

### 2.2 逆用四格

七殺、傷官、偏印、羊刃——可以顺用也可以逆用，但采用逆用时命格更高。

### 2.3 各格喜忌速查

| 格局 | 喜 | 忌 | 扶身首选 |
|------|-----|-----|----------|
| 正官 | 透财佩印 | 伤官/七殺/合官/刑冲/身弱 | 临官(禄) |
| 正财 | 身强用官 | 比劫/合财/刑冲/身弱 | 临官(禄)，不宜比劫 |
| 正印 | 官殺相生 | 财星/合印/刑冲/**身强** | 不忌身弱，忌身太强 |
| 食神 | 财星 | 偏印/合食/刑冲/身弱 | 比劫禄刃皆可，唯不宜印 |
| 七殺 | 食制/印化/身殺二停/刃驾殺 | 财殺齐出/合殺/刑冲/身弱 | 羊刃最佳 |
| 傷官 | 生财/佩印/伤尽 | 见官(无调候时)/刑冲/身弱 | 视调候而定 |
| 偏印 | 偏财+天月德 | 食神/羊刃/刑冲 | 身强透财即佳 |
| 羊刃 | 七殺 | 孤刃无依/双合/冲合并见 | 必须配七殺或正官 |

### 2.4 梁湘润的关键补充

**正官格**：用比肩扶身——可以，但比肩会制财，失去官星优点。用临官（禄）扶身——最佳。用印扶身——若官印根强则嫌过度。

**财格**：先财后印无碍，先印后财则不利。经曰"先财后印反成其福，先印后财恐增忧辱"。

**印格**：印格不忌身弱，反忌身强。印格+羊刃同在八字中——身体经常有病，或性格优柔不易合群。

**七殺格**：食神制殺——食神必须在前，七殺在后。食制与印化不可并见，否则印克食神，七殺无制。身殺二停的标准组合：甲寅日庚申月、丙午日壬子月。

**傷官格**：核心在调候而非格局。调候用财则伤官生财，调候用印则伤官佩印，调候用官则伤官见官亦佳。五行傷官中，金水/土金/木火三种命造较高，火土傷官最难处理（火炎土燥）。

## 三、取格优先级与特殊规则

### 3.1 取格优先级

1. 月柱天透地藏 — 第一优先
2. 月支透年时天干 — 第二优先
3. 月干坐根于年日时地支 — 第三优先
4. 地支三合三会取格 — 最后手段
5. 同类相聚勉认 — 无格可取时

双格同时成立时：同五行取偏（偏财>正财，七殺>正官，傷官>食神，偏印>正印）。不同五行则以月令及排位先后论别。

### 3.2 半合半刑的严格规则

- 半三合必须带子午卯酉，否则不算（申辰不算半三合水）
- 半合半刑不能取格
- 冲合并见时：先冲后合为吉，先合后冲为凶
- 经曰"先合后冲，祸患隐于吉中；先冲后合，吉祥来自冲刑"

### 3.3 拱夹法

三柱天干相同、地支相隔一字，可以"拱"出缺位的地支。

- 拱出的地支可以取格
- 拱出的地支可以与大运流年构成刑冲合会
- 拱的含义是"意料不及"
- 拱贵人——有意料不到之人相助
- 拱驿马——意料不到的升迁

### 3.4 本运年限精确划分

| 柱位 | 天干主 | 地支主 |
|------|--------|--------|
| 年柱 | 1-7.5岁 | 7.5-15岁 |
| 月柱 | 16-22.5岁 | 22.5-30岁 |
| 日柱 | 30-37.5岁 | 37.5-45岁 |
| 时柱 | 46岁至终 | — |

## 四、楊鶴朋用神新论：统计反推体系

这是全书后半部分最核心的思想，也是梁湘润认可收入的原因。

### 4.1 核心命题：大运用神 ≠ 流年用神

> "适合大运的用神绝对不可用在流年上，适合流年的用神亦不可用在大运上。"

- **大运重月令**（从节气月令演化），取法偏格局调候
- **流年重时支**（从出生时辰感应），取法偏统计验证
- 两者从不同角度演化而来，喜用绝对不可混淆共用

### 4.2 方法论：从结果反推用神

传统方法：理论推导 → 定用神 → 预测吉凶（理 → 用）
楊鶴朋方法：观察实际行运结果 → 分辨吉凶运 → 择出固定用神（用 → 理）

> "从既往的歲运分辨出吉运与凶运，而取用固定的用神，与扶抑、调候、通关有所不同，比较接近科学统计原则。"

### 4.3 五个反传统案例

这些案例说明传统用神取法与实际行运可能相矛盾：

1. **乙日生亥月（冬）**：调候取丙丁己午，但逢己午大运必定夫妻磨擦外遇
2. **乙生午月（夏）**：传统取癸水为用，但亥运（癸水长生）反为凶运
3. **丁生亥月逢子运**：亥为正官子为七殺，三会水，但不论官殺混杂——依运途显示，仍以正官纯清论
4. **己土生亥月**：穷通宝鉴取甲木，但地支无根则效用大打折扣，可用酉丑合化食傷吐秀替代
5. **辛日生亥月逢巳运**：冲提月令，传统大凶，但实际安然无恙且发财无数

## 六、102条歲运公式精选

以下为楊鶴朋102条中最有实战价值的条目：

### 身旺类
- 身旺食神制殺，逢七殺歲运则名利双辉（条1）
- 身旺帶伤官，逢财星歲运发财无数（条63）
- 身旺逢财运，娶妻贤能（条36）
- 身旺财入库，妻理家勤俭。逢歲运刑冲财库，发财无数（条41）

### 身弱类
- 身弱逢七殺运，容易贪心劳而无功，官司车祸等（条9）
- 身弱逢财运，娶妻败耗财产（条37）
- 财多身弱，常负债，接近女色事业惨败。逢比劫歲运反而事业蓬勃（条38）
- 身弱逢财歲运，流年再逢官殺，则官司是非多身体欠安（条40）

### 食伤类
- 食神为喜用，歲运逢偏印，无法成功发展，身体易有痔疮牙齿不良（条74）
- 歲运逢食伤，朋友合伙投资生意或学技术（条64）
- 时上伤官逢正印大运流年，勤奋向上且较会读书（条76）

### 女命类
- 坤命逢官殺歲，所认识交往的男人具稳重正直品德（条20）
- 坤命逢伤官歲运，所认识交往的男人较风流不上进（条21）
- 坤命伤官重行劫财羊刃歲运，家庭不幸夫败亡（条60）
- 坤命时柱食神逢歲运偏印，怀孕不吉（条83）

### 五行太过不及
- 土多金埋，金多水浊，水多木泛，木多火塞，火多土焦（条95）
- 湿木不生火，燥土不生金，寒金增水冷，火炎必土焦（条101）
- 燥土不制水，燥土不泄火（条102）

## 七、与徐乐吾体系的对比

| 维度 | 徐乐吾 | 梁湘润+楊鶴朋 |
|------|---------|----------------|
| 方法论方向 | 理→用（从理论推导实践） | 用→理（从实践统计反推理论） |
| 用神观 | 格局用神为主线，调候为补充 | 用神分五层，大运用神≠流年用神 |
| 论命入口 | 月令格局 → 成败救应 | 十神定位（速判50%）→ 格局 → 调候 |
| 形象哲学 | 情和气协、清浊真假、体用精神 | 无，偏实务统计 |
| 十干性情 | 十干实物说，详细展开 | 无 |
| 调候深度 | 五个技术系统，逐月逐干详解 | 给表不给思路 |
| 行运判断 | 格局成败救应+运能变格 | 诗词系统+102条公式+大运/流年分离 |
| 神煞态度 | 几乎不用 | 有限度采用，重天月德、桃花、孤寡 |
| 对"冲"的态度 | 冲多为凶 | 正气无刑，正位越冲越旺 |

### 互补关系

徐乐吾解决"为什么"——格局为什么成、为什么败、为什么这个用神比那个好。
梁湘润解决"怎么入手"——先看什么、后看什么、用什么速判。
楊鶴朋解决"实际验证"——理论推导的用神与实际行运不符时怎么办。

三者的实战使用顺序：
1. 排盘后，先用**十神定位法**速判50%吉凶（梁）
2. 查**穷通宝鉴**定调候方向（徐）
3. 定**格局成败救应**框架（徐·子平真诠）
4. 用**情和气协**审美标准精修判断（徐·滴天髓）
5. 行运判断：先用**格局变格**理论（徐），再用**大运/流年分离**交叉验证（楊）
6. 遇到理论与实际矛盾时，以**统计验证结果**为准（楊）

## 附录：出生地纳气差异

楊鶴朋举了一个极端案例：同年同月同日同时生的两人，一个台北一个台中，因"纳气"不同：
- 台北（纳气属木）：身弱，喜用火土
- 台中（纳气属土）：身强，喜用金水

喜用完全相反，职业、家庭、健康均有差异。这说明出生地的五行气场会影响八字取用。这个观点在其他命理书中几乎未见。
