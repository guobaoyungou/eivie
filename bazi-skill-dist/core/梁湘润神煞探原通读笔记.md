## 一、神煞的根本立场

### 1.1 神煞与格局的关系

梁的核心观点：**神煞不能取代格局，格局也无法覆盖神煞所指的全部吉凶。**

> "有一些八字，其吉凶祸福之运程，几乎与一般格局、用神、喜忌完全解释不通之情形发生。这可能就是受了神煞之影响。"

格局论"富贵贫贱"，神煞论"意外灾变"。两者各有所长：
- 火灾、车祸、官讼、伤残——格局推不出来，神煞有或然率
- 功名事业、财务兴衰——格局为主，神煞为辅

### 1.2 神煞的八种组合原理

1. **卦理**（如天罗地网取乾坤卦位）
2. **日月行度**（如天月德以三百六十五度分十二宫）
3. **奇偶方圆**（三合法、四生四旺四库）
4. **统计累积**（如"乙巳癸三干全，肢体有伤"）
5. **象形字义**（如甲丙丁壬为"平头"）
6. **佛伽博理**（如百日关、落井关等俗说）
7. **先天数理**（太玄数推驿马等）
8. **特殊排列组合**（如三奇贵人）

### 1.3 神煞吉凶的四大类

1. **论吉不论凶**：天乙贵人、天月德贵人
2. **可吉可凶**：劫煞、亡神、空亡
3. **论凶不论吉**：挂剑、天屠
4. **干支统计论凶**：乙巳癸三干全主伤残

### 1.4 神煞的使用层次

- 先天八字带神煞 → 一生基调
- 大运中遇神煞 → 十年隐忧
- 流年中遇神煞 → 该年具体凶兆

## 二、命理实用神煞（核心内容）

### 2.1 天火杀（火灾）

条件：四柱寅午戌全 + 天干有丙或丁 + 合计五个火字 + 四柱无水
流年：行火旺处或合去水的大运流年

### 2.2 戳肢杀（后天伤残）

按月取日时天干双见：正月甲甲、二月乙乙、三月戊戊...
大运流年忌忌神旺支、刑合之期

### 2.3 破杀（少年多灾连累家长）

年支与时支对应：卯见午、丑见辰、子见酉、未见戌

### 2.4 天刑杀（一生多病）

年支配时干：子丑年忌乙时、寅年忌庚时、卯辰年忌辛时、巳年忌壬时、午未年忌癸时、申年忌丙时、酉戌年忌丁时、亥年忌戊时
**时干为甲、己者免此煞。**

### 2.5 电灵狂杀（卒然临难）

月配时：正七月生子时、二八月生寅时、三九月生辰时、四十月生午时、五十一月生申时、六十二月生戌时

### 2.6 死病符（痼疾）

歲後一辰，带冲。如巳年后一辰为辰，辰见戌冲 → 日时有戌者带痼疾。

### 2.7 官符杀（官讼是非）

太歲前五辰落在日时 → 一生有官灾
干支顺行五辰落在日主 → 多为受害者而招官非

### 2.8 挂剑杀（刀刃血光）

四柱巳酉丑申全，或四柱有局部而大运流年会齐。
兼带元辰、亡神、白虎者更重。

### 2.9 天屠杀（疑难杂症）

日支与时支对照：子日午时、丑日亥时、寅日戌时、卯日酉时、辰日申时、巳日未时（反向亦同）

### 2.10 自缢杀（自杀倾向）

年支与时支对应（五行反业）：子年酉时、丑年午时、寅年未时、卯年申时、辰年亥时、巳年戌时（反向亦同）

### 2.11 冲天杀（少年夭折）

条件极严格：
1. 年月冲 + 日时冲（双冲）
2. 双冲**不在同一组**（四生/四旺/四库须分属两组）
3. 纳音五行须相克（时克年之纳音）

### 2.12 天罗地网

**天罗**：戌亥。**地网**：辰巳。四字齐全为天罗地网。
- 男怕天罗（日时见戌亥，火命为忌）
- 女怕地网（日时见辰巳，水命为忌）
- 四柱本身齐全 → 一生灾不能息
- 大运流年补齐 → 该年大凶
- 出生日数 = 灾年数（初一生一年灾，初二生二年灾...初三十生三十年灾）

**或然率高于贵人** —— 梁个人特别重视此煞。

### 2.13 干支统计规则（极重要的"无名神煞"）

这些出自明代统计，虽无神煞名称，但有实际验证价值：

| 条件 | 主事 |
|------|------|
| 壬癸见酉亥多 | 因酒惹事 |
| 乙巳癸三干全 | 四肢眼目可能被伤 |
| 乙癸干同见丑 | 主富而少寿 |
| 丁辛二干同见巳 | 伤父母 |
| 乙辛二干同见未 | 主富贵中夭 |
| 庚壬二干同见戌 | 主富贵中夭 |

## 三、子平法常用神煞补充

### 3.1 孤辰寡宿

并非只论"鳏寡离婚"。古义：老而无夫曰寡，幼而无父曰孤。

沈芝独特论法（年支对时支互冲 = 孤鸾）：
少年欠儿。日时克之，活不过房，必克妻。

广录论法：二重孤寡 = 克妻害子、少六亲、不聚财、多生女。

### 3.2 劫煞十六种

梁列出劫煞的十六种细分组合——不是见劫煞就凶：
- 聚宝劫煞（五行与日主不配 → 富）
- 宜催劫煞（临官+空亡 → 贵人）
- 福禄劫煞（身强劫弱+贵人 → 安康）
- 刀枪劫煞（干坐七殺克日主 → 真凶）
- 空闲劫煞（兼带孤寡 → 宗教人士）

### 3.3 亡神十五种

同样细分：亡神带马星 → 三灾；亡神带祿马贵 → 发达；亡神带咸池 → 凶...

### 3.4 驿马的真正含义

**古注从未说驿马是"出远门"。**

古经原文：驿马者，五行有为待之气强名也。

壽總曰：
- 生旺 → 气韵凝峻，通变超时，多声望
- 死绝 → 有头无尾，漂泊不定
- 与禄同乡 → 福力优游
- 与煞相冲 → 离乡背井

驿马五配（三命通会原表）：
- 寅午戌马在申：甲申=截路空亡马、戊申=福星伏马、壬申=大败马、丙申=大败马、庚申=天昌马
- 其余三组类推

### 3.5 咸池（艺术性格）

咸池在旺宫 → 智慧艺术秉奇高秀
咸池在衰宫+空亡+七殺 → 偏激个性、不信鬼神

### 3.6 羊刃作为神煞（非格局）

梁特别指出：羊刃与建禄在明代都是神煞，到清代才被列为"格局"。

实务要点：
- 甲见乙卯为刃最怪
- 庚逢酉为刃最怪
- 壬逢子为刃最怪
- 时干逆对日支为刃 → 夫妻不和
- 朝元羊刃（年支对日干/时干之刃位）→ 不利父母

## 四、天乙贵人的陰陽分论

梁引郭景纯的原始推导：

**阳贵**起于先天坤（子），甲加子 → 己用子为贵人，不居午（天空）
**阴贵**起于后天坤（申），辛加酉 → 丙用酉为贵人，不居寅

戊为天罡/河魁，贵人不临。

## 五、协纪辨方书提要（择日神煞，参考用）

梁节录了清代协纪辨方书的完整神煞体系，作为理解命理神煞来源的参考。

核心框架：建除十二神（建满平收黑，除危定执黄，成开皆可用，闭破不相当）

重要择日概念：
- 歲德（年之吉神，陽干为主）
- 歲德合（年之吉神，陰干为辅）
- 天德月德（命理择日通用）
- 三煞（劫煞+灾煞+歲煞，最凶之组合）
- 黄道黑道（十二神的吉凶日，青龙明堂金匮天德玉堂司命为黄道吉日）

## 六、江湖术语探原与实务论命

### 6.1 清代巾行二十种

梁详列清代民间算命的二十种形式：大金（六壬）、小运（拆字）、八分卦（奇门）、追子巾（鸟卦）、量巾（量手）、弯巾（少年敲板）、斩盘（面相）、旺巾（开口算命）、八黑（笔批流年）...

### 6.2 四字诀（率擸扣代）

梁传授的"业余论命"实务心法：

- **率**：率出来宾真正关心什么事（用试探性语句）
- **擸**：奉承，测出对方喜好加以讲美
- **扣**：重语扣住人心，令对方不再追问
- **代**：收场送客的学问

### 6.3 根苗花果实务示范

梁以乙未、甲申、丁卯、庚子女命为例，完整演示：

**八字一瞥**：一财得所、自坐偏印、七殺桃花 → 婚姻不美满

**已知因**（来宾是亲友介绍的离婚女士+富翁遗孀）→ 与八字完全符合

**开口论命**："你早婚不吉，癸亥年有夫妻灾难，属于有自由职业之女性。中年有帮夫运，相夫教子之贤妻。虽有中途镜[?]之分，无碍一生富足之命。"

**三阶段**：
1. 观听感（来宾身份、言谈气质）
2. 落实局（八字核心推理的客切表达）
3. 收关局（补充发问+送客收场）

## 七、神煞实战排查流程（适用任何八字）

排八字后，依次检查以下凶煞是否成立：

1. **天火杀** — 四柱寅午戌全 + 天干丙丁 + 五火 + 无水
2. **天刑杀** — 年支与时支特定组合（午年忌癸时等）
3. **电灵狂杀** — 月份与时支特定组合（七月生子时等）
4. **天罗地网** — 辰巳戌亥四地支齐全
5. **干支统计** — 特定双干/三干组合（庚壬同见戌、乙巳癸三干全等）
6. **自缢杀** — 年支与时支特定对应关系（午年未时等）
7. **冲天杀** — 年月冲 + 日时冲 + 不同组 + 纳音克
8. **孤辰寡宿** — 按年支查（如午年孤辰在申，寡宿在辰）

排查原则：

- 各项独立检查，分别记录"犯/不犯"
- 神煞只是辅助信号，不能作为主要断法（陈素庵原则）
- 命带孤辰主性格独立、不依靠六亲；命带寡宿主孤独感强
- 神煞排完后必须回到正五行格局综合判断
