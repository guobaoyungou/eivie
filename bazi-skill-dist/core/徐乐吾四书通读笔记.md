## 一、徐乐吾的体系定位——他到底在干什么

徐乐吾自述"生平所敬服膺者为子平真诠、穷通宝鉴暨滴天髓三书"。他做的事是：**用穷通宝鉴的调候经验去注解子平真诠的格局理论，再用滴天髓的形象哲学去融通二者。** 造化元钥是穷通宝鉴的精简版，提供五个技术系统的架构。

他的核心信念：格局法和调候法不是两种方法，是一件事的两面。格局是骨架（what），调候是血肉（when），滴天髓是灵魂（why）。

但徐有一个致命弱点：**理论强于实战**。子平真诠评注中多处被林注（后人批注）直接指出取用错误，尤其在从格判断和身强身弱的分辨上频繁失误。读者必须带着批判眼光读他的评注，他的价值在体系构建和理论阐发，不在具体批命。

## 二、五个技术系统（造化元钥独有）

造化元钥把穷通宝鉴的调候用神归纳为五个相互循环的技术系统，这是其他三本书没有的架构：

### 系统一：甲庚丁三干循环
- **甲木**日主用丁火（伤官）+ 庚金（偏官）
- **丁火**日主用甲木（正印）+ 庚金（正财）
- **庚金**日主用丁火（正官）+ 甲木（偏财）
- 三者互为用神，缺一不可，构成闭环
- 立论表象：甲用丁庚→木火通明，聪慧文采；庚用丁甲→利器成材，成就事业；丁用庚甲→引灯添薪，忠孝双全

### 系统二：丙辛用壬
- **丙火**为太阳，必用壬水（江湖之水）映衬→"壬水辅丙"
- **辛金**为珠玉，必用壬水淘洗→"壬水洗淘法"
- 丙辛都不用癸。丙见癸是不晴不雨，辛见癸是雨露不足以淘洗

### 系统三：乙用癸丙
- **乙木**为花草，必须癸水（雨露）滋养根基 + 丙火（阳光）照暖
- 寒冬腊月重丙火，夏令炎热重癸水
- 乙木不用壬水——壬如洪水漂浮花草

### 系统四：戊己共用甲丙癸
- **戊土**为高亢堤岸，首重甲木疏辟（犁锄开垦），次丙火暖土，次癸水润土
- **己土**为田园湿土，首重癸水润泽（灌溉发生），次丙火暖土，次甲木疏土
- 戊己虽同用甲丙癸，侧重不同：戊先甲，己先癸
- 季节权变：冬令戊己皆先丙（解冻为急），夏令皆先癸（润燥为急）

### 系统五：壬癸用戊辛丙（复杂系统）
- 壬癸水的用法随季节变化最大：
  - 春水衰弱→要庚辛发源 + 戊土堤防
  - 夏水干涸→要庚辛发源（存亡关头）
  - 秋水旺涝→要戊土堤防（止其泛滥）
  - 冬水严寒→要丙火照暖（解冻方现自由）

### 实战意义
这五个系统回答了"什么日元在什么月份用什么"的第一层筛选。排盘后先定日元属于哪个系统，然后查该系统的调候链是否完整。**用神链完整且透干→上命，有一二藏支→中命，全缺→下命。**

### 五个系统之间的交叉关系

五个系统不是孤立的，彼此有交叉：
- 甲庚丁循环中的**庚金**，同时也是壬癸系统中"发源"的关键（庚生壬水）
- 甲庚丁循环中的**甲木**，同时也是戊己系统中"疏土"的关键
- 丙辛用壬中的**壬水**，同时也是壬癸系统的主角
- 乙用癸丙中的**丙火**，同时也是冬令所有天干调候的共用神

**实战时的判断顺序**：先确定日元属于哪个系统 → 查该系统的主要用神链 → 检查交叉系统是否有冲突 → 若有冲突则看月令季节决定优先级。

如庚金日元生于冬月：属于甲庚丁循环（需丁火煅炼），但冬令需丙火解冻（存亡级调候）。丙丁同为火，但作用不同——丙是太阳暖局，丁是炉冶成器。穷通宝鉴的回答是：**三冬庚金，丁为主丙为佐**，先丁后丙。即成器为主，暖局为辅。但若丁火全无，则丙火独挑两担——既暖局又代行煅金之功，只是效果不如丁。

## 三、"十干实物说"的推理方法（穷通宝鉴核心思维）

穷通宝鉴论命的底层逻辑不是抽象的五行生克，是**物理常识比拟**：

| 天干 | 实物 | 推理逻辑 |
|------|------|----------|
| 甲 | 栋梁之木 | 必须庚金斧斤劈削方"成材" |
| 乙 | 花草之木 | 必须癸水雨露+丙火阳光方"向荣" |
| 丙 | 太阳之火 | 必须壬水江湖映衬方"辉煌" |
| 丁 | 炉冶之火 | 必须甲木薪柴劈开方"光明永驻" |
| 戊 | 堤岸之土 | 必须甲木犁耕+丙火暖实+癸水润泽方"载物" |
| 己 | 田园之土 | 必须丙火暖+癸水润方"生机盎然" |
| 庚 | 顽矿之金 | 必须丁火炉冶煅炼方"成器" |
| 辛 | 珠玉之金 | 必须壬水淘洗方"现宝气" |
| 壬 | 江湖之水 | 必须戊土堤防方"奔行千里不泛滥" |
| 癸 | 雨露之水 | 必须辛金润泽发源+丙火升腾方"泽于苍生" |

**关键：这套推理是比喻式的，不是逻辑式的。** 你不能用"庚克甲"来推出庚金需要甲木——推理方向是从"实物功能"出发，不是从"五行生克关系"出发。这是穷通宝鉴独特的认识论基础，也是它容易被误读的原因。

### 实物说与格局法的矛盾

十干实物说会和子平真诠的格局法产生矛盾。举例：

庚金日元生七月（申月），格局法说"月令建禄，别取用神"——可以用财、用官、用食伤。但穷通宝鉴说"七月庚金司令，刚锐已极，**专用丁火煅炼**，次取甲引丁光"——不管什么格局，丁火是第一需要。

这两套说法如何统一？徐乐吾的答案是：**调候定方向，格局定层次**。穷通宝鉴告诉你七月庚金"必须见丁火"，这是方向；子平真诠告诉你丁火在你八字里充当什么角色（是官星？是用神？还是相神？），这是层次。方向对了层次低，也有衣禄；方向错了层次高，也是枉然。

### 实物说的三条立论特色（造化元钥总结）

穷通宝鉴有三个独特的立论角度，与其他命书不同：

1. **气候优先于月令正五行**：正月建寅，正五行当论甲丙戊，但穷通宝鉴说"正月甲木，初春尚有余寒"——以实际气温（余寒）为第一考量，不以寅中甲木当令为第一考量
2. **重阳干轻阴干，重偏神轻正神**：甲木不离庚金（偏官），丙火不离壬水（偏官），都是阳干喜偏官偏财。"阳干力量大特别喜偏官偏财，尤利做大事业；阴干力量弱，喜相生利守成"
3. **十干实物说**：把十干当成实物来推理（如上表），而不是当成抽象符号

## 四、徐乐吾对滴天髓原文的关键修正

### 4.1 "阳支阴支"改为"阳干阴支"

滴天髓原文"阳支动且强，阴支静且专"，徐乐吾认为原文有误字，应为"**阳干动且强，阴支静且专**"。理由：天干动而有为（吉凶顺而易见），地支静以待用（吉凶隐而经年）。这个改动影响对干支作用力大小的判断。

### 4.2 合与化的严格区分

任铁樵注滴天髓时常以"合"即"化"论。徐反复强调：**合与化不同。合而不化当论其合，合而能化当论其化。今人见合皆作化论，殊误。** 化必须得时得地成方成局，否则只论合不论化。

### 4.3 体用二字的精确定义

前人论体用混用不清。徐在滴天髓补注中做了最清楚的区分：
- **体**：月令提纲是体（八字的"环境"）。偏旺格以全局气势为体。
- **用**：提纲所喜之神为用。相神即用之辅佐。
- **精神**：配合有情有力，即有精神。精神不是另一个概念，是体用配合得宜的外在表现。

他的原话："体用分清，即不致发生错误。"——先辨体（月令是什么），再定用（月令需要什么），最后看精神（配合是否到位）。

### 4.4 对"得时不旺失时不弱"的深化

子平真诠原文已有此论，但徐在评注中做了重要补充：

**旺衰与强弱是两回事**：
- **旺衰**看月令——得时为旺，失时为衰
- **强弱**看党众——比印多为强，克泄多为弱
- 故有"旺而弱"（得时但克泄重）和"衰而强"（失时但比印多）

如甲木生春（旺），但干庚辛支酉丑，金之党众远超木之势力→旺而弱。甲木生秋（衰），但干甲乙支寅卯，比印重叠→衰而强。**日主如此，喜用忌神皆同此论**。

**通根比干头重要**：
- 天干得一比肩，不如支中得一墓库（如甲逢未、丙逢戌）
- 得二比肩，不如得一余气（如乙逢辰、丁逢未）
- 得三比肩，不如得一长生禄旺（如甲逢亥子寅卯）
- "比劫如朋友之相扶，通根如室家之可住；干多不如根重"

这直接影响实战中判断身强弱的权重分配——**先数支根，后数干助**。

### 4.5 滴天髓中"天覆地载、上下情协"的八字审美观

徐在补注中反复以"天覆地载"评价八字好坏。其含义：
- **天覆**：天干所透之神，地支有根承载
- **地载**：地支所藏之神，天干有字透出
- 干透支藏，天覆地载，方为有根有力
- 如四辛卯、四丙申——干支不相通，天不覆地不载，虽一气亦弱

具体标准：
- 天干透出之字在支中有禄旺→最强
- 天干透出之字在支中有长生→次强
- 天干透出之字在支中有余气墓库→又次
- 天干透出之字在支中毫无根气→浮露无用

徐评康熙帝命（甲午·戊戌·戊申·丁巳）："此造无形之中自有一种浑厚之气...八个字中一字不可改易，地位次序一字不能互换...此为真正上格局。"——强调的不是某个用神多好，而是**整体结构的不可替换性**。

### 4.6 滴天髓补注中"顺逆"的完整说法

原文只有"顺逆不齐也，不可过于顺...不可过于逆"。徐的补注给出了完整的操作指南：

**我生之局（食伤）→引其性情**：
- 食伤太旺不可克制（如戊土太旺不可用甲木），应顺而泄之（用庚金泄土之气）
- "强水得木方泄其势，强木得火方化其顽，强火得土方止其焰"——元理赋的精髓

**生我之局（印绶）→顺乎气势**：
- 印旺成局不可逆，须顺其方向
- 如印旺身旺的格局，不可用财破印，而应用食伤泄秀——顺着印→身→食伤→财的方向流通

**此即为什么有些格局"用官不可见伤、用印不可见财"的深层原因——不是死规矩，是气势不可逆。**

## 五、子平真诠评注中的实战规则（只有逐行读才能提取）

### 5.1 位置决定吉凶——生克先后法

同是正官逢财逢伤，位置不同结果截然相反：

| 格式 | 列式 | 结论 |
|------|------|------|
| 伤先财后 | 丁年·己酉·甲日·戊辰 | 财旺生官，伤官之气泄于财，官不受损 |
| 财先伤后 | 戊年·辛酉·甲辰·丁卯 | 财生官后遭伤破，官最终受损 |
| 有间隔的保护 | 癸年·甲寅·丙日·戊时 | 官能生印，戊不能越甲合癸，大贵 |
| 无间隔的合破 | 癸年·戊午·丙日·甲午 | 戊癸相合，格大破 |

**规则提炼**：
1. 喜神在年月、忌神在日时 → 先得后失
2. 喜神在日时、忌神在年月 → 先失后得
3. 有间隔之字能解二者冲突 → 间隔为救应
4. 日元本身之合不作合去论 → "我之官，是我合之"

### 5.2 闲神的隐藏作用

徐反复强调：八字中喜忌用神之外，还有闲神。闲神平时无作用，但行运时若闲神被冲合引动，可以转顺为逆或转逆为顺。**同一格局因闲神不同，运的吉凶完全不同。**

如薛相公命（甲申·壬申·乙巳·戊寅），甲运帮身吉。但如果年柱甲申改为己酉，行甲运则合劫破印，即不美。"所谓因闲神之配合而喜忌不同也。"

### 5.3 干支行运的分别

核心十二字：**"运以方为重"，但"行干而不行支"亦有之。**

- 庚申辛酉：干支同气，金运无疑
- 丙子丙申：火不通根，金水之气为重
- 甲午乙未：木火同气，但午火旺木受泄
- 丙生子月亥年：行丙丁帮身吉，行巳午冲旺水反增水势→行干不行支
- 甲生酉月辛透：行申酉官植根，行庚辛混煞重官→行支不行干

**实战口诀**：干支同气全吉全凶；干支不同气，干到位不到方（力微），支到方不到位（力重）。

### 5.4 行运成格变格

这是子平真诠中最难的部分。徐举了一个极好的实例：

姚文敷造（辛未·甲午·丙申·戊戌）：月令阳刃，但丙临申位旺而不旺。平时食神泄秀为用，气势不足。至寅运，寅午戌三合齐，身旺泄秀，阳刃用食格局突然完成。寅运中一跃为两淮盐运使。

**规则**：原局格局"成而未全"，逢运补足那一个字，格局突然完整，富贵陡至。但仅此运五年（或十年），过后依然如故。

### 5.5 子平真诠格局成败的完整程式

通读全书48章后，可以把格局成败的逻辑总结为一张完整的决策树：

```
月令取格 → 格神确定
  ↓
格局成否？
  成：格神有力 + 相神配合 + 无忌破格
    → 成中有败？（带忌神）
      → 有救应？（合去/制住忌神）
        → 败中有成 → 贵格
        → 无救应 → 破格
    → 无忌 → 纯格 → 视有情无情、有力无力分高低
  败：格神受伤/无相神
    → 有因败得成？（因会合变化而忌转为喜）
      → 有 → 变格取贵
      → 无 → 贫贱
```

**八格的完整成败条件**：

| 格局 | 成 | 败 | 救应 |
|------|----|----|------|
| 正官 | 官逢财印+无刑冲 | 官逢伤克/合去/杂煞 | 印制伤护官/合去煞 |
| 财格 | 财旺生官/食生财+身强 | 财轻比重/财透煞 | 食化劫/官制劫/合煞 |
| 印格 | 印轻逢煞/官印双全/身印两旺用食伤 | 印轻逢财/身强印重透煞 | 劫去财/合财存印 |
| 食神 | 食神生财/食带煞无财弃食就煞 | 食逢枭夺/食生财露煞 | 财制枭/弃食就煞 |
| 七煞 | 身强煞逢制 | 煞逢财无制 | 食制煞生财/合煞 |
| 伤官 | 伤生财/伤佩印+伤旺印有根/伤带煞无财 | 非金水见官/生财带煞/佩印而伤轻身旺 | 财化伤生官/印制伤 |
| 阳刃 | 透官煞+财印相随 | 无官煞 | 煞刃相持+印缓气 |
| 建禄月劫 | 透官逢财印/透财逢食伤/透煞逢制伏 | 无财官+透煞印 | 合去忌神/化劫为生 |

### 5.6 各格取运的统一原则

通读正官至杂格取运各章后，总结出取运的统一原则：

1. **用神宜助不宜伤**——最基本
2. **相神更不可伤**——"伤相甚于伤用"
3. **身轻则取助身，官轻则取助官**——偏弱方需补
4. **身旺印重，财运反吉；身弱财旺，印运反吉**——此消彼长
5. **闲神被冲合引动，可转顺为逆/转逆为顺**——运中最难预见
6. **官煞不可混**——运中见混亦忌
7. **成格之运与变格之运，福祸尤重于平常喜忌**——运能成格是大起，运能变格是大落
8. **干支同类不两行**——如丁壬一合一帮，效果不同；同为寅运，午戌在局则会火，子辰在局则会水

### 5.7 伤官五种格局的取运差异（最复杂的格）

伤官格变化最多，子平真诠专门分了多种子类，取运各异：

| 子格局 | 用法 | 运喜 | 运忌 |
|--------|------|------|------|
| 伤官生财 | 身强泄秀生财 | 财食 | 官煞（除金水调候外） |
| 伤官佩印 | 身弱制伤扶身 | 官煞（生印）、印 | 财（破印） |
| 伤官兼用财印 | 财多带印/印多带财 | 助弱的那一方 | 助强的那一方 |
| 伤官用煞印 | 身弱煞生印化伤 | 印旺 | 财（破印党煞） |
| 伤官用官（金水） | 调候为急见火 | 财印 | 食伤再泄 |
| 伤官带煞 | 以伤驾煞 | 印（去食卫煞）；煞重运喜食伤 | 财（化食生煞） |

**关键心法**：伤官格先看身强身弱 → 身强看有财无财 → 身弱看有印无印 → 最后看月令季节（金水？木火？火土？土金？水木？）→ 季节决定调候需求 → 调候需求可能翻转常规喜忌。

## 六、穷通宝鉴的隐含规则（只有逐行读才能发现的模式）

### 6.1 调候优先级的三层结构

通读十干×十二月全文后，发现调候取用有固定的优先级：

**第一层：存亡级**（缺此无用）
- 冬令一切天干→丙火解冻（"非丙不暖"反复出现）
- 夏令壬癸水→庚辛发源（"无庚不能蓄源"）
- 春木→不可无水亦不可无火（水火既济）

**第二层：品级级**（有此升格）
- 甲木→庚金劈甲+丁火通明
- 庚金→丁火煅炼
- 辛金→壬水淘洗
- 戊土→甲木疏土

**第三层：配合级**（此为辅佐）
- 用丁制庚→不可无甲引丁（甲庚丁循环）
- 用壬辅丙→不可无庚生壬（源远流长）
- 用癸润己→不可无辛生癸（雨露有源）

### 6.2 十干取用的同异规律

通读后发现，十干在不同月份的取用有一个隐含的对称结构：

| 月份 | 甲木 | 乙木 | 关系 |
|------|------|------|------|
| 正月 | 丙癸并用 | 丙癸并用 | 同 |
| 二月 | 庚金为主 | 丙癸并用 | 异——甲用庚劈，乙不受金克 |
| 夏月 | 先丁次庚（通明） | 先癸后丙（润泽） | 完全不同 |
| 秋月 | 先丁次庚（成材） | 先丙后癸（护花） | 完全不同 |
| 冬月 | 庚为要丙次之 | 专用丙火 | 同中有异 |

**规律**：阳干甲丙戊庚壬多喜偏官偏财（阳克阳，利做大事业）；阴干乙丁己辛癸多喜正印正官（阴生阴，利守成）。穷通宝鉴的第二个立论特点正是**"重阳干轻阴干，重四凶神轻四吉神"**。

### 6.3 "真神得用"的判断标准

穷通宝鉴反复说"丙癸齐透，科甲有准"、"甲庚两透，科甲有准"。通读后提炼判断标准：

1. **透干**：用神必须天干透出（藏支减半）
2. **通根**：透干之字必须地支有根（无根浮露不真）
3. **当令**：用神得月令之气最旺（月令休囚减力）
4. **无伤**：用神不被合去、克伤、冲破
5. **有配**：主要用神外还有辅佐配合

五条全满→大贵；满四条→贵；满三条→有衣禄；满一二条→平常；全不满→下命。

### 6.4 穷通宝鉴的断语体系

穷通宝鉴的断语有一套固定用语，理解这套用语才能快速读懂每段：

| 断语 | 含义 |
|------|------|
| 科甲有准 | 正途出仕（进士/举人），大贵 |
| 选拔/贡监 | 非正途的较低功名 |
| 异途显达 | 不由科举而获官位（如军功、捐纳） |
| 衣衿/秀才 | 最低级功名 |
| 有富贵 | 有钱有地位但不确定多少 |
| 衣禄充足 | 温饱无忧但无大发 |
| 常人 | 普通百姓 |
| 僧道孤贫 | 最差，无妻无子无财 |
| 奸诈刻薄 | 性格有缺陷 |
| 富而不贵 | 有钱无官职 |
| 贵而不富 | 有官职清贫 |
| 先贫后富 | 用神在时柱（晚年得力） |
| 先富后贫 | 用神在年月（早年享用）而时上忌神 |
| 妻贤子肖 | 财星和官煞有情得力 |

### 6.5 穷通宝鉴中"一例看/分别看"的规则

通读后发现穷通宝鉴频繁使用"一例"和"不同"来标记哪些月份可以合并看、哪些必须分开看：

**可以合并看（"一例"）**：
- 甲木正二月一例（但阳刃格二月独有）
- 甲木五六月一例
- 丙火四五月一例（专用壬水）
- 丁火三秋一例
- 戊土正二月一例
- 辛金正二月一例

**必须分开看**：
- 所有天干的**三月**（辰月土旺）都要单独看，不能和一二月合并
- 所有天干的**六月**（未月）都要按大暑前后分两段
- 所有天干的**九月**（戌月土旺）都要单独看
- 所有天干的**十二月**（丑月）都要按上下半月分看

**底层逻辑**：辰戌丑未四季月是转折点，气候交脱之际，用神最易变化，所以必须独立分析。寅卯/巳午/申酉/亥子两个月之间气候连续，可以合看。

### 6.6 穷通宝鉴的"妻子"断法

穷通宝鉴每段末尾常有"用神妻子同前"之语，其规则是：

- **用官煞者**：财为妻，食伤为子（因财生官，食制煞）
- **用丙火者**：木妻火子
- **用癸水者**：金妻水子
- **用甲木者**：水妻木子
- **用庚金者**：土妻金子

规则很简单：**生用神者为妻，用神所生者为子**。与滴天髓"喜神有意傍天财"一脉相承。

## 七、滴天髓补注中的独特判断法

### 7.1 "殿印相生、元机暗藏"

徐在滴天髓补注中反复使用一个判断法：**看支中是否暗藏殿印相生的链条。**

如戊寅日坐长生，寅中甲丙戊三字——丙戊殿印相生（丙火印生戊土），甲木偏官也在其中。这是"绝处逢生"——日元虽临绝地，坐下暗藏生机。

类似的：
- 甲申日：申中庚壬戊→庚金偏官有壬水化为印
- 癸丑日：丑中己辛癸→辛金正印暗生日元
- 庚午日：午中丁己→丁火正官有己土印化

**这四柱（戊寅甲申癸丑庚午）被徐称为"六十花甲中特别之日"**，生者每多大贵格局。

### 7.2 寒暖燥湿四诊法

滴天髓第三篇"征验"中，徐提出八字也有寒暖燥湿之病：
- **过寒**：金水冬月无火→"金寒水冷"→性格阴沉消极
- **过暖**：木火夏月无水→"火炎木焚"→脾气暴躁短寿
- **过燥**：火土旺月无水→"土焦金脆"→皮肤病、脾胃病
- **过湿**：水木冬月无火→"水冷木浮"→关节病、肾病

**诊法**：先看月令在哪个季节，再看四柱干支的寒暖燥湿倾向，最后看有无调和之神。

### 7.3 滴天髓的格局分类法（与子平真诠不同）

子平真诠以月令八格为纲，滴天髓以"形象"为纲，分类方式完全不同：

**独象**：一行独旺（曲直、炎上、稼穑、从革、润下五格）→ 宜行化地（泄秀），忌逆旺势

**偶象（两神成象）**：
- 相生之象（如木火、金水）→ 顺其相生方向取用
- 相克之象（如金木、水火）→ 须通关之神
- 身煞两停（煞刃格）→ 用印缓气

**从象**：日主无根从强势（从财、从煞、从儿/食伤）→ 顺从势为用，忌比劫帮身

**化象**：天干合化成功（五种化气格）→ 化神喜旺地，忌还原

**全象**（正格）：三行以上，体用精神俱全 → 即子平真诠的八格法

**从象的真假判断**（徐在补注中的核心贡献）：
- **真从**：日主完全无根，满局一片克泄 → 地位自高，行从化旺乡即贵
- **假从**：日主微有根苗或印有点根 → 行从化旺乡亦可取富贵，但遇根苗运则还原，一时之顺逆
- **不能从**：日主有根而财官旺 → 仍按正格论

徐引《滴天髓》原文："真从之象有几人？假从亦可发其身。"——真从极罕见，假从常见，但假从在得运时同样可以富贵。

### 7.4 "情和气协"——判断八字整体品质的终极标准

徐在滴天髓补注的干支总论中，把"上下贵乎情和，左右贵乎气协"提到最高位置。

**情和**（上下）：日干与日支、年干与年支、各柱干与支之间的配合关系
- 天地合（如甲子见己丑，干支上下都合）→ 精气团结
- 天合地刑（干合支冲）→ 有情有碍
- 干支相生（如丙寅、壬申）→ 天覆地载

**气协**（左右）：年月、月日、日时之间的流通关系
- 四柱顺生（如年木→月火→日土→时金）→ 始终不悖，福寿最厚
- 四柱逆克（年克月、月克日）→ 气势不顺，多坎坷
- 联珠（地支相连如寅卯辰）→ 气聚一方

**徐论康熙帝命造时说的"八个字中一字不可改易、地位次序一字不能互换"，就是情和气协的极致。** 一般命造做不到这种完美，但可以拿这个标准去衡量：情和的程度 × 气协的程度 = 格局高低的底线。

### 7.5 滴天髓论行运的独特视角

滴天髓论运与子平真诠有一个重要的区别视角：

**子平真诠论运**：从格局出发——运助格则吉，运破格则凶
**滴天髓论运**：从气势出发——运顺气势则吉，运逆气势则凶

二者大部分情况下结论一致，但在偏旺格局中可能不同。如一方专旺格，子平真诠可能说"无官煞制刃，格败"，但滴天髓说"气势偏旺，顺而不逆，行泄秀之运即吉"。

**徐的折中**：正格用子平真诠的方法论运，偏旺格和从格用滴天髓的方法论运。这是最稳妥的做法。

## 八、徐乐吾的错误与局限（读者必须知道）

### 8.1 从格判断频繁失误

子平真诠评注中，林注多次指出徐把从格当正格、把正格当从格：
- 朱家骅造（癸巳·丁巳·丁卯·丙午）：徐取癸水抑制日元，林注指出火成气势必然从火
- 赵知府造（丙午·庚寅·丙午·癸巳）：徐说化印为劫、弃印就财官，林注指出财官无根应从旺
- 某造身弱财食并旺，徐说"身强财食并旺"——完全相反

### 8.2 用神取用时常牵强

穷通宝鉴评注中，徐常把调候规则硬套到不适用的八字上。韦序中直言："乐吾先生虽然学问渊博，但其评注间有牵强附会之处，不免乎硬凑而离题也。"

### 8.3 纳音格局偶尔干扰正论

造化元钥中"甲子乙丑为死木，甲戌乙亥为活木"的分类，徐自己也承认"参考纳音言也，理有未合"。

### 8.4 子平真诠评注中的具体错误汇总

逐行读时标记的林注（后人校正），列出最典型的几处：

| 原造 | 徐的判断 | 林注纠正 | 核心错误 |
|------|----------|----------|----------|
| 朱家骅（癸巳·丁巳·丁卯·丙午） | 日元太旺，取癸水抑制 | 火成气势必然从火，癸水是忌神 | 把从格当正格 |
| 赵知府（丙午·庚寅·丙午·癸巳） | 化印为劫弃印就财官 | 财官无根应从旺格 | 把从格当正格 |
| 谢阁老（己未·壬申·戊子·庚申） | 身强财食并旺 | 无火土无原神当身弱论 | 身强弱判断反了 |
| 曾毓隽（乙亥·癸未·己亥·辛未） | 己土通根用神太弱扶之 | 无火土无原神应弃命从杀 | 把从格当正格 |
| 某辛丑造（辛亥·庚子·甲辰·乙亥） | 以印为用 | 冬水旺极水泛木浮当从水论 | 印旺不等于用印 |

**共同模式**：徐倾向于按正格扶抑法看命，对从格的识别能力不足。阴干从势的案例尤其容易误判。

### 怎么用徐乐吾
- **用他的体系框架**：五个技术系统、格局成败救应的完整程式、调候三层优先级
- **不用他的具体判断**：具体八字的身强弱、用神取舍，须自行分析验证
- **带着林注读**：子平真诠评注中凡有"林注"处，先看林注，再看徐注
- **穷通宝鉴评注中的命例分析部分谨慎参考**：韦千里序言直言"牵强附会"，命例判断不可全信
- **造化元钥和滴天髓补注的理论部分可信度最高**：这两本是徐的理论主场，阐发精到

## 十、通读后最深刻的一句话

徐乐吾在子平真诠评注末尾自述："予用是术以历试，但有贫贱，并无富贵。"——他承认自己的理论在实战中未必全验。然后他说："然要之极贱者多，不得不观其人物以衡之。"

这句话的意思是：**命理学能排除大量的"不可能"，但不能精确定位"必然"。** 这和认识论中梁湘润说的"六七成或然律"完全一致。

徐乐吾最大的贡献，不是他断命有多准，而是他把三本互不相干的经典串成了一个体系，让后人有了一个"先查什么、再查什么、最后查什么"的清晰路径。这个路径本身，比任何单一的断命方法都有价值。
