## 一、教学体系总架构

### 1.1 四阶段一年制

| 阶段 | 时长 | 内容 | 教材 |
|------|------|------|------|
| 第一 | 3个月 | 背诵四组规则 + 十神性格六亲 + 女命 | 子平基础概要 + 讲义一 |
| 第二 | 3个月 | 格局顺逆用 + 日主五行喜忌 + 调候 | 讲义一后半 + 讲义二前半 |
| 第三 | 3个月 | 实务论命 + 六十甲子 + 流年 | 讲义二 + 古今图书集成选读 |
| 第四 | 3个月 | 细批终身 + 大运流年流月 + 史观 | 讲义二后半 + 金不换大运 |

### 1.2 授课次序（与传统截然不同）

传统次序：排八字 → 身强身弱 → 格局喜忌 → 用神 → 流年
梁的次序：排八字 → **十神性格** → **十神六亲** → **女命** → 偏印偏财 → **坐引论** → 格局常识 → 调候 → 大运 → 流年

核心逻辑：先让学员"开口能说"（十神定位可速断性格六亲），再回头补理论（格局喜忌调候）。

### 1.3 为什么先讲偏印偏财，不先讲正官

梁的理由极其实务：
- 正官格一上来就是"傷官见官、官殺混杂"，初学者听了只会更糊涂
- **偏印格**在八格中推理最简单：喜偏财+天月德，忌食神
- **偏财格**最贴近来宾关心的事：父亲、钱财、女人
- 两者在性格描述上最有代入感，学员不觉枯燥

### 1.4 七种论命层次

梁把论命方式分为七个层次，层层递进：

1. **样本八股**（滴天髓式）：身强制财，先财后印，取戊为用
2. **传统诗赋**（三命通会赋文）：三甲分明天上光，甲日申逢喜运中
3. **中和素描**（五行十神泛论）：比肩通根，食傷过多变伤官
4. **书生说理**（业余层次）：通根年限量度 + 刑冲合会全局平衡
5. **对答就事论事**（实务初阶）：来宾问什么答什么，不展开理论
6. **代客统筹规划**（实务高阶）：为来宾制定未来三年行动方针
7. **综合现实流年**（终极层次）：以上全部融合，但以眼前流年为第一优先

**实务论命只需第5-7层。** 前4层是"说理"，业余研究的层次。

## 二、十神实务八法（讲义一核心框架）

每个十神按统一框架论述，此为梁独创的教学结构：

### 2.1 框架

1. 五行性 → 陰陽异同决定有情无情
2. 性格定位 → 天干二见或过多时的人格特征
3. 六亲单向生克 → 该十神生什么、克什么，代入六亲
4. 四柱年限 → 在年/月/日/时各有不同的实际人生意义
5. 坐引禄绝 → 同柱坐旺坐绝、引向其他三支的旺绝
6. 格局喜忌 → 该十神成格时的顺逆用
7. 天干坐地支十神 → 干支互换后的十神对十神
8. 实务周边 → 天透地藏的形势、过多的判定标准

### 2.2 "过多"的统一判定标准

任何十神的"过多"定义：
- 天干2见 + 地支1见 = 性格上有决定性力量
- 地支三合或三会 + 天干再透1个 = 吉凶上有明显凶兆

### 2.3 比肩与劫财的实务差异

| 维度 | 比肩 | 劫财 |
|------|------|------|
| 阴阳 | 同性相斥 | 异性情切 |
| 性格 | 不稳定、自我否定、举棋不定 | 精明刚强、得理不让、外柔内刚 |
| 无利害时 | 设身处地、引小人图利 | 客气保持距离、微妙打量他人 |
| 涉利害时 | 答应了吃亏也承担 | 慣诺、吃亏立刻反悔 |
| 论命取舍 | 有理也肯让人（除非大事） | 小事也不让，引小人来攻击 |
| 扶身质量 | 如二十块薄板拼合，撑不久 | 如原木，有情相助但性烈 |

### 2.4 偏印性格三层次

梁对偏印的性格描写极为精细：

**对内**：同性相生如"不良公务员上班"——能省就省、能推就推、应付式帮助。主性格孤僻、间断性反复不定、模棱两可、忸怩作态。

**对外**：简短语刺、陈义太高、高雅式固执。做对了冷漠孤芳自赏，失意时椎心之压力与沉闷。

**对亲属**：男命对外人沉默、对自己人斤斤计较。女命最宜艺术宗教、不适大家庭。

偏印格的唯一吉路：偏财 + 天月德贵人。

梁的人情化比喻："一位写字很好的人，需要部长级的人说他写得好才能成名书法家——所以一定要天月德贵人。定价五百元的字没有价值，不定价一百万也未必写——所以要偏财。"

### 2.5 偏财性格精要

偏财过多者的十五条特征（梁的实战描写）：

- 讲原则不在细节，不斤斤计较
- 经常保持笑容，不因小事勃然变色
- 肯先吃虧，但不是圣贤之道，是"输得起"的心胸
- 作事有战略思想，围标工程愿付日息五万
- 肯帮助朋友，不是讲义气，是"一时高兴"的真正施恩不望报
- 涉商业的话不可尽信——太过乐观，说谎如翻书
- 日常不按常规，比自己大十岁或小二十岁的女人都可以交往
- 财务弹性极强，破产六七次仍能东山再起
- 最标准的偏财格——代理商
- 女命偏财格是孝女

## 三、全方位坐引论（讲义一第八节，实务核心）

这是梁湘润实务论命的真正核心方法。

### 3.1 坐

同一柱中天干与地支的关系。

**坐禄**：天干十神在地支有临官位——该十神在此年限极旺
**坐绝**：天干十神在地支为绝位——该十神似有似无，最低标准
**坐十神**：将地支换为祿位天干，天干十神对地支十神互论

例：正财己坐卯（乙劫财祿位）——以劫财透正财论，先劫后成。
反之：劫财乙坐卯（乙劫财祿位）坐正财——正财有一部分被劫，先劫后成。

### 3.2 引

天干对照其余三个地支的禄绝关系。

例：年干庚偏财坐申（庚禄在申，15岁前父运好）→ 引月支寅（庚绝在寅，20-30岁父败或父子不和）→ 引日支申（又为庚祿，35-45岁偏财事业成功）→ 引时支戌（庚在戌为衰，晚年财力渐退）

### 3.3 拱

三柱天干相同，地支相隔一字，拱出缺位地支。

拱的特色：
- 一定拱子午卯酉（四正位）
- 拱出的字不在明处，代表"意料之外"
- 拱出的字最怕被四柱另一字正冲——致命之伤
- 拱出的字可以取格、可以与大运流年构成刑冲合会

### 3.4 蓋头截脚

天干坐绝（或地支是天干之绝位）：
- **截脚**：偏财甲申——甲绝于申，最低标准的偏财
- **蓋头**：七殺庚寅——庚绝于寅，寅中甲财旺但被庚蓋住

比喻：进得了电影院，但坐在第一排最偏边的座位。

### 3.5 疊旺疊绝

地支二见同一旺位或绝位：
- 日主地支二个帝旺（羊刃）→ 过份自信，凶多吉少
- 日主地支二个绝 → 一生难有佳绩
- 女命正官七殺分别在地支居旺位 → 二度婚姻

## 四、女命论法（讲义一第七节）

梁把女命列为基础教材的重点，因为女性来宾占大多数。

### 4.1 十项推理框架

1. 夫星子星地支轻重（月支看夫，时支看子）
2. 日主旺相（禄刃印俱全 = 无婚姻）
3. 官殺混杂（天干/地支/月柱坐专位三种形态）
4. 三德天乙文昌（天月德增福，天乙解难但女命忌多，文昌主艺文）
5. 十神疊遇（同一十神年月干干透坐 = 定位吉凶）
6. 反克（比劫反克财、伤官反克官、财反克印、印反克食伤）
7. 五行缺一（女命最忌缺官殺夫星）
8. 神殺五元次生克
9. 刑冲空合会
10. 四生四旺四库

### 4.2 女命日主"出局"标准

祿+刃+印俱全 = 无婚姻：

| 日主 | 出局条件（四柱地支同时有） |
|------|--------------------------|
| 甲 | 寅、卯、子 |
| 乙 | 寅、卯、子 |
| 丙 | 巳、午、卯 |
| 丁 | 巳、午、寅 |
| 戊 | 巳、午、午 |
| 己 | 巳、午、午 |
| 庚 | 申、酉、巳 |
| 辛 | 申、酉、巳 |
| 壬 | 亥、子、酉 |
| 癸 | 亥、子、申 |

### 4.3 天乙贵人女命特论

一座天乙是好命，二座天乙心不定，三座天乙反为下。

原因：天乙贵人是"解难"之神，女人向男人借钱只有后遗症。一个贵人嫁给他就好，二三个贵人则"扯不清"。

### 4.4 官殺混杂三种形态

**天干明见**：年月混 = 30岁前感情阻折；月时混 = 体弱多病（中间隔日柱15年，不构成道德问题）；年时混 = 头胎子女有伤

**地支禄位**：天干无官殺，地支有申酉 = 暗中之事，很少人知道

**月柱专位**：月柱正官坐七殺 = 先有对象后再有另一位

## 五、三论并行（讲义一第十一节）

### 5.1 财官论

出处：三命通会·金玉赋·元机赋。重地支不重天干。

核心：四柱但有一支是财官当令地支，即构成"有用"。一个地支平均管八年，一般市民有八年顺运就能买房安家。

实例：庚日主见卯（乙正财禄位）在年 = 出身富有；在月 = 少年得佳伴；在日 = 中年得佳妻十五年好运；在时 = 晚年得佳妻

### 5.2 格局论

出处：子平真诠。重天透地藏，重顺逆用。适合"功名富贵"层次。

### 5.3 十神定位论

出处：梁湘润综合三命通会散赋编成。不谈理论，直接答案。

三者适用对象：
- 士大夫问功名 → 格局论 + 调候
- 百姓问妻财子禄 → 财官论 + 十神定位
- 短期问事（一两年内）→ 流年刑冲合会

## 六、讲义二核心：实务提要

### 6.1 生旺库凶年（第一优先，不问格局强弱）

观察四柱有无二字属同组：
- 四生：寅申巳亥
- 四旺（四正）：子午卯酉
- 四库：辰戌丑未

四柱已有二字，大运流年补齐 → **该年作凶论，不问任何格局强弱**

天罗地网（戌亥辰巳齐全）比四生四旺四库更凶，涉及寿元。

拱合也算：年月暗拱巳，四柱有辰戌 → 大运或流年逢丑即补齐四库。

复式：四柱同时有子午卯酉和寅申巳亥 → 一生至少八次凶流年。

### 6.2 格与局的根本区别

这是讲义二最核心的理论贡献：

**格**：天透地藏取出。用**顺法**——不可反克。
**局**：三合三会取出。用**逆法**——必须反克。

例：甲生酉月，酉即辛正官 → 正官格（顺用）→ 忌伤官
甲生地支申酉戌三会金 → 七殺格（逆用）→ 要食制或印化

**三合三会之格局律作逆用**，不分正偏。这是子平真诠（论格）和造化元钥（论局）的根本区别。

具体规则：
- 生我者三合三会 → 偏印格
- 我生者三合三会 → 伤官格
- 我克者三合三会 → 偏财格
- 克我者三合三会 → 七殺格

### 6.3 四见成格

天干地支合计四个同五行：
- 四见透天干 + 有旺支 → 成格（等同三合三会）
- 四见透天干 + 无旺支 → 也可成格
- 四见不透天干 + 有旺支 → 不成格，只是"多根"（终身难除的性格特征）
- 四见不透天干 + 无旺支 → 多根，终身难除

四见多根的四种性格答案：
- 食伤多根：刚柔不定，想出头又不甘落后
- 财多根：容易到手但只能办小事，多女人缘
- 印多根：没有定性的文艺爱好，时常改变
- 七殺多根：一生犯小人（其实是借用小人之心），女命感情困扰

### 6.4 拱与夹的精确区分

| 类型 | 条件 | 拱出内容 | 稳定性 |
|------|------|----------|--------|
| 三合拱 | 同天干二柱，地支差二位 | 子午卯酉 | 不稳定（只留一孤支） |
| 三会夹 | 同天干二柱，地支差一位 | 三会成局 | 较稳定（留二余位） |
| 夹角拱 | 同天干二柱，地支隔一位但跨越不同三会 | 角位地支 | 最不稳定（如路人顺手扶一把） |

**三合拱夫**（女命）：可以有等同夫妻之待遇，但无名份（暗夫同居）
**夹角拱夫**（女命）：感情不会维持太久

### 6.5 金不换大运（讲义二后半核心）

原理：
- 调候用神取**天干**
- 金不换大运取**地支**
- 两者不矛盾，互补

金不换大运三大特色：
1. 只取月令地支为主格（甲生酉月 = 正官格）
2. 不必每步运都论，只指出一生中最佳与最坏的三步运
3. 除吉凶外，还直接论寿夭

大运地支的层次不是五行生克，而是十二生旺库：
- 庚日主行寅运：不论寅是甲木（偏财），而论庚绝在寅 → 此运中另有新的意外成功
- 大运天干坐绝于地支：该天干十神无效（甲申运，甲绝于申 → 偏财无效益）

### 6.6 流月规则

流月与四柱**无**刑冲合会关系（五年一循环，重复率太高）
流月与流年**无**刑冲合会关系（固定组合，全人类相同）

流月吉凶只看：
1. 月运同甲子（与大运伏吟）
2. 月运反吟大运
3. 月运双合大运
4. 月运拱合四柱+大运

**通时气**：秀在时支 = 晚年始亨通。
**地支禄位即天干十神代词**：卯即乙，申即庚——不另说明。

### 7.4 壬日主的特殊发现

## 十二、《子平母法大流年判例》通读笔记

> 262页/8661行，行卯出版社，初版戊午年(1978)，此为丙子年(1996)第八版改编
> 梁湘润最重要的实务流年教材，真人实事批命实例
> 核心主张：流年重流月，不重身强弱；财官派与用神派是两套系统

### 12.1 三大命学体系（梁的终极分类）

| 体系 | 核心方法 | 适用对象 | 代表 |
|------|----------|----------|------|
| **指标命学** | 格局顺逆用+调候+身强弱 | 士大夫论功名（三榜进士、亿万富翁） | 子平真诠、滴天髓、穷通宝鉴 |
| **生态命学** | 十二生旺库性格+十神生态+神煞 | 百姓妻财子禄 | 三命通会散赋、巫咸经 |
| **财官派** | 地支财官禄位+刑冲合会 | 一般中产（一宅一车） | 元机赋、巾箱经验 |

**关键认识**：
- 指标命学是"说理"层次，能论方向但不能论细节
- "身弱不能托财官"此语是指司道大员的官、亿万资财之财。一般中产，身弱身强并不是唯一必须重视的事项
- 财官派：四柱地支中但有一支二支是日主财官禄位，不遇刑冲，即可论为一宅一车之命
- **三者不是非此即彼，是层次递进：指标定上限，生态定性格，财官定日常**

### 12.2 流年实务核心法则

**流年与四柱的两种看法**（梁的核心区分）：

| 方向 | 程序 | 适用 |
|------|------|------|
| 四柱→流年 | 先定格局用神，再推大运流年 | 细批终身（业余研究） |
| **流年→四柱** | 先看眼前流年，以大运为桥梁回看四柱 | **实务论命（专业应用）** |

**流年批导七大法则**：

1. **大运是桥梁**：大运干支对四柱的刑合冲拱，形成"半成型架构"，流年在此架构中触发吉凶
2. **流月是重点**：流年干支定大方向，流月干支定具体月份吉凶。不能批流月就不能批流年
3. **流月三个关键**：月运同途（流月=大运）、月运反吟（流月冲大运）、月运双合（流月合大运）
4. **八字喜忌≠流年喜忌**：好八字未必年年好，好流年不必好八字才有
5. **伏吟反吟优先**：流年与四柱同干支=伏吟，干支双冲=反吟，二者不利六亲，非横破财
6. **连续看2-3年**：不能只看当年，须兼顾前年和明年的干支趋势
7. **日犯岁君最重**：日干克流年天干=下犯上，灾殃必重

### 12.3 双轨命学——十二生旺库性格论（独立体系）
