# 使用说明

## 这是什么

中华传统命理 AI Agent skill。支持 Claude Code / Codex CLI / OpenCode 等。

能帮你排八字 + 排紫微 + 看大运流年流月，融合徐乐吾/梁湘润/袁树珊/韦千里四家方法 + 紫微斗数中州派。

包含 17 部公共领域典籍 + 9 篇方法论笔记（含当代著作通读笔记）+ 4 个名人案例 + 28 人索引 + 预构建知识图谱。

---

## 怎么装（30 秒）

**macOS / Linux：**
```bash
cd bazi-skill-dist
bash setup.sh   # 选 1 = 轻量，选 2 = 含 LightRAG
```

**Windows：**
```cmd
cd bazi-skill-dist
setup.bat
```

完事。重启你的 AI Agent（Claude Code / Codex / OpenCode 等）就能用了。

（高级 LightRAG 模式安装详情见 `core/rag-usage.md`）

---

## 怎么用

直接在 AI Agent 里说话。例如：

```
帮我排八字：阳历 1990 年 5 月 15 日 上午 10 点出生，男，北京
```

```
我是戊土日主生在申月，调候用神是什么？查穷通宝鉴
```

```
帮我看 2026 年的流年流月，四家方法对照
```

```
我的紫微命宫紫微七杀在亥，怎么看？引陆斌兆讲义
```

```
对比一下我和邵逸夫的盘
```

Claude 会自动按十五步流程排盘，引用典籍出处，标注每一步推理。

---

## 关键铁律（Claude 会自动遵守）

1. **手动排盘**——脚本排盘出过严重错误（半合误判等），全部手动推理
2. **半合必带子午卯酉**——巳丑、寅戌、卯未都不算半合
3. **禁止反推**——不能拿你已知的事实凑解释
4. **不知道就说不知道**——禁止蒙猜

## 不能用来做什么

- 不能替代医生（健康问题去医院）
- 不能替代律师（法律问题找律师）
- 不能做日内交易决策（命理粒度不到这种级别）
- 不能歧视任何人（命理不评判好坏）

详见 ETHICS.md。

---

## 出问题怎么办

**Claude 没有自动加载 skill？**
检查 `~/.claude/skills/bazi-ziwei-mingli/SKILL.md` 是否能读到。

**典籍查不到内容？**
直接说"查 books/八字/穷通宝鉴.md 里关于戊土的部分"，Claude 会用 Grep 工具找。

**想用 LightRAG 高级模式？**
看 SKILL.md 第十.五节 B 部分，5 步装好。
