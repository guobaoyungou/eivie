## 1. 商家端使用指南

### 1.1 系统配置

#### 1.1.1 登录系统
1. 访问商家后台登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 进入系统主界面

#### 1.1.2 导航菜单
登录后，在左侧菜单中找到"AI旅拍"模块，包含以下子菜单：
- 商家配置管理
- 提示词模板管理
- 素材管理
- 作品管理
- 订单管理
- 统计报表

---

### 1.2 商家配置管理

#### 1.2.1 新增商家配置
**步骤**:
1. 点击"AI旅拍 - 商家配置管理"菜单
2. 点击"新增配置"按钮
3. 填写配置信息：
   - 商家名称：您的商家名称
   - 可灵AI AccessKey：从可灵AI平台获取
   - 可灵AI SecretKey：从可灵AI平台获取
   - 监控路径：本地文件夹路径（支持多目录，用分号分隔）
     - Windows示例：`D:\photos\monitor;E:\videos\monitor`
     - Linux示例：`/var/www/monitor;/var/www/videos/monitor`
   - 模型名称：kling-v1（默认）
   - 生成模式：std（标准）/pro（专业）
   - 画幅比例：16:9（默认）
   - 视频时长：5（秒）
   - 自动上传：是/否
4. 点击"保存"按钮

**注意事项**:
- AccessKey和SecretKey请妥善保管，不要泄露
- 监控路径必须是绝对路径
- Windows路径使用反斜杠`\`，Linux路径使用正斜杠`/`
- 支持配置多个可灵AI账号，系统会自动负载均衡

#### 1.2.2 编辑商家配置
**步骤**:
1. 在配置列表中点击某个配置的"编辑"按钮
2. 修改配置信息
3. 点击"保存"按钮

#### 1.2.3 删除商家配置
**步骤**:
1. 在配置列表中勾选要删除的配置
2. 点击"删除"按钮
3. 确认删除操作

---

### 1.3 提示词模板管理

#### 1.3.1 新增图生视频模板
**步骤**:
1. 点击"AI旅拍 - 提示词模板管理"菜单
2. 点击"新增模板"按钮
3. 填写模板信息：
   - 模板名称：如"风景-公园"
   - 模板类型：图生视频
   - 提示词：一个美丽的女孩在公园里散步，阳光明媚，背景有绿树和草地
   - 负向提示词：模糊、低质量
   - 模型名称：kling-v1
   - 生成模式：std
   - 画幅比例：16:9
   - 视频时长：5
   - 排序：1
4. 点击"保存"按钮

**提示词编写技巧**:
- 详细描述场景、人物、动作、环境
- 使用正向词汇，避免负面描述
- 指定风格、色调、氛围
- 可以包含镜头语言（如：特写、远景、俯拍）
- 控制视频长度在50-200字之间

#### 1.3.2 新增文生视频模板
**步骤**:
1. 点击"新增模板"按钮
2. 选择模板类型为"文生视频"
3. 填写模板信息：
   - 模板名称：如"古装-古镇"
   - 模板类型：文生视频
   - 提示词：一个穿着古装的女孩在古镇里拍照，古色古香，背景有古建筑
   - 模型名称：kling-v1
   - 生成模式：std
   - 画幅比例：9:16
   - 视频时长：10
   - 排序：2
4. 点击"保存"按钮

#### 1.3.3 新增视频特效模板
**步骤**:
1. 点击"新增模板"按钮
2. 选择模板类型为"视频特效"
3. 填写模板信息：
   - 模板名称：如"风格转换"
   - 模板类型：视频特效
   - 特效场景：style_transfer
   - 模型名称：kling-v1
   - 排序：3
4. 点击"保存"按钮

**支持的特效场景**:
- `style_transfer` - 风格转换
- `super_resolution` - 超分辨率
- `color_correction` - 色彩校正
- 更多特效场景请参考可灵AI官方文档

#### 1.3.4 编辑模板
**步骤**:
1. 在模板列表中点击某个模板的"编辑"按钮
2. 修改模板信息
3. 点击"保存"按钮

#### 1.3.5 删除模板
**步骤**:
1. 在模板列表中勾选要删除的模板
2. 点击"删除"按钮
3. 确认删除操作

#### 1.3.6 导入模板
**步骤**:
1. 点击"导入模板"按钮
2. 粘贴模板JSON数据
3. 点击"导入"按钮

**导入JSON格式**:
```json
{
  "template_name": "导入模板",
  "template_type": "image2video",
  "prompt": "提示词内容",
  "negative_prompt": "负向提示词",
  "model_name": "kling-v1",
  "mode": "std",
  "aspect_ratio": "16:9",
  "duration": "5",
  "sort": 1
}
```

### 1.4 素材管理

#### 1.4.1 手动上传素材
**步骤**:
1. 点击"AI旅拍 - 素材管理"菜单
2. 点击"手动上传"按钮
3. 选择文件（支持图片和视频）
   - 图片格式：jpg, jpeg, png
   - 视频格式：mp4
   - 文件大小限制：10MB
4. 点击"上传"按钮
5. 等待上传完成

**注意事项**:
- 图片建议分辨率：1920x1080或更高
- 视频建议时长：5-30秒
- 文件命名建议：使用有意义的名称，便于管理
- 上传后素材会自动进入AI生成队列

#### 1.4.2 查看素材列表
**步骤**:
1. 进入"AI旅拍 - 素材管理"页面
2. 查看所有上传的素材
3. 可以按上传时间、素材类型、上传方式筛选
4. 支持分页查看

**素材信息**:
- 素材名称
- 素材类型（图片/视频）
- 上传方式（手动/自动）
- 上传来源
- 文件大小
- 上传时间
- 关联任务数
- 关联作品数

#### 1.4.3 删除素材
**步骤**:
1. 在素材列表中勾选要删除的素材
2. 点击"删除"按钮
3. 确认删除操作

**注意事项**:
- 删除素材会同时删除关联的任务和作品
- 删除操作不可恢复，请谨慎操作

#### 1.4.4 刷新监控状态
**步骤**:
1. 点击"刷新监控"按钮
2. 等待刷新完成
3. 查看最新的监控状态

**监控状态信息**:
- 监控路径
- 最后检查时间
- 监控文件数量
- 上传成功数
- 上传失败数

### 1.5 作品管理

#### 1.5.1 查看作品列表
**步骤**:
1. 点击"AI旅拍 - 作品管理"菜单
2. 查看所有AI生成的作品
3. 可以按创建时间、支付状态、价格筛选
4. 支持分页查看

**作品信息**:
- 作品名称
- 作品类型（视频/图片）
- 预览图（带二维码）
- 价格
- 支付状态（已支付/未支付）
- 浏览次数
- 下载次数
- 创建时间
- 关联订单

#### 1.5.2 设置作品价格
**步骤**:
1. 在作品列表中找到要设置价格的作品
2. 在价格输入框中输入新价格（单位：元）
3. 按回车键或点击"更新"按钮
4. 查看价格更新成功提示

**定价建议**:
- 基础作品：5-10元
- 精品作品：10-30元
- 特殊作品：30-50元
- 可以根据景区定位、作品质量调整价格

#### 1.5.3 删除作品
**步骤**:
1. 在作品列表中勾选要删除的作品
2. 点击"删除"按钮
3. 确认删除操作

**注意事项**:
- 删除作品不会删除已支付的订单记录
- 删除操作不可恢复，请谨慎操作

#### 1.5.4 刷新作品列表
**步骤**:
1. 点击"刷新作品"按钮
2. 等待刷新完成
3. 查看最新的作品信息

## 2. 游客端使用指南

### 2.1 微信授权

#### 2.1.1 扫码授权
**步骤**:
1. 使用微信扫描商家提供的二维码
2. 进入游客端H5页面
3. 点击"扫码授权"按钮

5. 授权成功后自动跳转到作品列表页面

**授权信息**:
- 微信昵称
- 微信头像
- 会员ID
- 访问令牌（Access Token）

**注意事项**:
- 首次使用需要授权
- 授权后30天内有效
- 可以在个人中心查看授权信息

### 2.2 查看作品

#### 2.2.1 浏览作品列表
**步骤**:
1. 访问作品列表页面
2. 查看所有AI生成的作品
3. 可以按创建时间、支付状态筛选
4. 支持分页查看

**作品展示**:
- 网格布局展示
- 每个作品显示预览图（带二维码）
- 显示作品名称、价格、支付状态
- 未支付作品显示"购买"按钮
- 已支付作品显示"查看"和"下载"按钮

**步骤**:
1. 在作品列表中点击某个作品的预览图
2. 进入作品详情页面

**作品详情信息**:
- 大尺寸预览图
- 作品名称
- 作品类型（视频/图片）
- 视频时长（如果是视频）
- 视频尺寸
- 创建时间
- 浏览次数
- 下载次数
- 支付状态
- 操作按钮

**操作按钮**:
- 购买：未支付作品显示
- 查看：已支付作品显示

### 2.3 选择作品

#### 2.3.1 单选作品
**步骤**:
1. 在作品列表中点击某个作品的"选择"按钮
2. 作品卡片显示选中状态
3. 可以继续选择其他作品

**选中状态**:
- 作品卡片边框变色
- 显示"已选择"标记
- "创建订单"按钮显示选中的作品数量

#### 2.3.2 多选作品
**步骤**:
1. 在作品列表中点击多个作品的"选择"按钮
2. 每个作品卡片显示选中状态
3. "创建订单"按钮显示选中的作品数量
4. 可以取消选择

**批量操作**:
- 全选：一键选择所有作品
- 取消全选：取消所有选择
- 批量购买：一键购买所有选中的作品

### 2.4 创建订单

#### 2.4.1 创建订单
**步骤**:
1. 选择一个或多个作品
2. 点击"创建订单"按钮
3. 查看订单创建成功提示
4. 点击"去支付"按钮

**订单信息**:
- 订单号：AV + 时间戳 + 随机数
- 作品数量
- 订单总金额
- 作品列表

**注意事项**:
- 订单创建后30分钟内需要完成支付
- 超时未支付订单会自动取消
- 订单创建后作品会锁定给当前用户

### 2.5 支付购买

#### 2.5.1 微信支付
**步骤**:
1. 在支付页面选择"微信支付"
2. 点击"立即支付"按钮
3. 查看微信支付二维码
4. 使用微信扫描二维码
5. 在微信中完成支付

**支付流程**:
1. 发起支付请求
2. 生成支付二维码
3. 用户扫码支付
4. 支付成功后跳转到作品详情页面
5. 作品状态更新为已支付
6. 发送支付成功通知

**支付成功后**:
- 作品状态更新为已支付
- 作品自动保存到相册
- 发送微信模板消息通知
- 可以查看和下载作品

#### 2.5.2 支付宝支付
**步骤**:
1. 在支付页面选择"支付宝"
2. 点击"立即支付"按钮
3. 跳转到支付宝支付页面
4. 在支付宝中完成支付

**支付流程**:
1. 发起支付请求
2. 生成支付宝支付链接
3. 用户跳转到支付宝
4. 在支付宝中完成支付
5. 支付宝回调通知服务器
6. 作品状态更新为已支付

#### 2.5.3 余额支付
**步骤**:
1. 在支付页面选择"余额支付"
2. 点击"立即支付"按钮
3. 查看支付结果

**支付流程**:
1. 检查会员余额是否足够
2. 扣除相应金额
3. 记录余额变动
4. 更新订单状态为已支付
5. 更新作品状态为已支付
6. 发送支付成功通知

**余额不足提示**:
- 显示"余额不足"错误提示
- 提示充值
- 提供充值入口

#### 2.5.4 支付成功
**支付成功后**:
- 显示"支付成功"提示
- 自动跳转到作品详情页面
- 作品状态更新为已支付
- 作品自动保存到相册
- 发送支付成功通知

**文件位置**:

**步骤**:

**路径示例**:
- 单目录：`D:\photos\monitor`
- 多目录：`D:\photos\monitor;E:\videos\monitor`
- Linux路径：`/var/www/monitor`

#### 3.1.3 配置商家ID
**步骤**:
1. 记录商家的商家ID
2. 在启动监控程序时传入商家ID

**启动命令**:
```bash
# Windows
php AivideoMonitor.php <商家ID>

# Linux
php AivideoMonitor.php <商家ID>
```

### 3.2 启动监控

#### 3.2.1 Windows系统
**步骤**:
1. 打开命令提示符（CMD或PowerShell）
2. 进入监控程序目录
3. 执行启动命令
4. 观察监控程序输出

**启动命令**:
```cmd
cd C:\AivideoMonitor
php AivideoMonitor.php 1
```

**监控输出**:
- 显示商家ID
- 显示监控路径
- 显示开始监控时间
- 检测到新文件时显示文件名
- 上传成功时显示成功信息
- 上传失败时显示失败信息和重试次数

#### 3.2.2 Linux系统
**步骤**:
1. 打开终端
2. 进入监控程序目录
3. 执行启动命令
4. 观察监控程序输出

**启动命令**:
```bash
cd /opt/aivideo
php AivideoMonitor.php 1
```

**后台运行**:
如果需要在后台持续运行，可以使用nohup或screen：
```bash
# 使用nohup
nohup php AivideoMonitor.php 1 &
```

### 3.3 常见问题

#### 3.3.1 监控程序无法启动
**问题**: 监控程序无法启动

**解决方案**:
1. 检查PHP版本是否为7.4.33+
2. 检查监控程序文件权限
3. 检查商家ID是否正确
4. 查看监控路径是否正确

#### 3.3.2 文件上传失败
**问题**: 文件上传一直失败

**解决方案**:
1. 检查网络连接
2. 检查服务器API是否正常
3. 检查文件大小是否超过限制（10MB）
4. 检查文件类型是否支持
5. 查看上传接口地址是否正确

#### 3.3.3 文件重复上传
**问题**: 同一个文件被重复上传

**解决方案**:
1. 系统会自动检测已上传的文件
2. 已上传的文件不会被重复上传
3. 如果发现重复上传，可以忽略

#### 3.3.4 监控程序崩溃
**问题**: 监控程序意外崩溃

**解决方案**:
1. 使用后台运行方式（nohup或screen）
2. 查看崩溃日志
3. 设置定时任务自动重启
4. 联系技术支持

#### 3.3.5 上传速度慢
**问题**: 文件上传速度很慢

**解决方案**:
1. 检查网络带宽
2. 检查服务器性能
3. 减小文件大小
4. 考虑使用CDN加速
