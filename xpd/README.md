# AI旅拍选片端Web系统

## 概述

AI旅拍选片端是部署在商户门店大屏设备上的展示系统，实时展示AI生成的旅拍照片和视频，吸引游客关注并引导扫码购买。

## 功能特点

- ✅ 实时轮播展示AI生成的照片和视频
- ✅ 支持多种展示模板切换
- ✅ 头像列表导航，中间位置高亮
- ✅ 自动生成二维码引导扫码购买
- ✅ 自动刷新数据，保持内容新鲜
- ✅ 7x24小时稳定运行
- ✅ 门店独立URL，数据隔离
- ✅ 演示数据支持，无数据时自动展示示例

## 访问方式

### 1. 门店独立URL

每个门店拥有独立的展示URL：

```
https://域名/xpd/index.html?aid={平台ID}&bid={商家ID}&mdid={门店ID}
```

**参数说明：**
- `aid`: 平台ID（必填）
- `bid`: 商家ID（必填）
- `mdid`: 门店ID（可选，不填则展示商家所有门店数据）

### 2. 后台管理入口

商家可在后台"系统 > 门店管理 > 编辑门店"页面中：
- 查看选片URL
- 复制链接
- 预览效果
- 查看二维码
- 选择展示模板

## 部署说明

### 1. 文件权限

确保Web服务器对目录有读取权限：

```bash
chmod -R 755 /www/wwwroot/eivie/xpd/
```

### 2. Nginx配置

确保静态文件正确映射：

```nginx
location /xpd/ {
    root /www/wwwroot/eivie;
    index index.html;
    try_files $uri $uri/ /xpd/index.html;
}
```

### 3. 数据库迁移

已自动执行，门店表新增字段：

```sql
ALTER TABLE `ddwx_mendian` 
ADD COLUMN `xpd_template` VARCHAR(50) DEFAULT 'template_1' COMMENT '选片端展示模板';
```

## 使用指南

### 商户操作流程

1. **配置门店**
   - 登录后台 > 系统 > 门店管理
   - 编辑门店信息
   - 选择展示模板

2. **获取URL**
   - 在门店编辑页面查看"选片URL"
   - 点击"复制链接"获取URL
   - 点击"预览"查看效果
   - 点击"二维码"获取二维码

3. **大屏部署**
   - 在门店大屏浏览器中打开URL
   - 设置浏览器全屏模式（F11）
   - 建议使用Chrome或Edge浏览器

4. **日常运营**
   - 系统自动刷新数据，无需手动操作
   - 如需切换模板，重新打开URL即可
   - 建议定期检查设备网络连接

### 游客使用流程

1. 在门店拍摄人像照片
2. 等待AI生成结果（约5-30秒）
3. 在大屏上查看生成的照片和视频
4. 扫描二维码查看详情和购买

## 常见问题

### Q1: 页面一直显示"正在加载数据..."怎么办？

**A:** 检查以下几点：
1. 确认网络连接正常
2. 确认URL参数正确（aid、bid必填）
3. 确认该门店已上传人像照片
4. 查看浏览器控制台是否有错误信息
5. **新：**如果没有真实数据，系统会自动使用演示数据展示

### Q1.5: 页面显示"演示模式"是什么意思？

**A:** 当没有真实数据时，系统会自动使用演示数据进行展示，方便预览效果。
- 演示数据包含5组人像和11张AI生成照片
- 页面顶部会显示红色提示条（★秒后消失）
- 系统会在60秒后自动重试获取真实数据
- 详细说明请查看：`DEMO_DATA.md`

### Q2: 轮播不流畅，有卡顿现象

**A:** 建议：
1. 使用Chrome或Edge浏览器
2. 检查设备性能，建议使用较新的设备
3. 检查网络带宽，建议100M以上
4. 减少后台运行的其他程序

### Q3: 二维码不显示

**A:** 可能原因：
1. 该人像尚未生成二维码
2. 二维码已过期
3. 等待系统自动生成（切换到下一组时重新检查）

### Q4: 如何切换展示模板？

**A:** 
1. 登录后台 > 系统 > 门店管理 > 编辑门店
2. 在"展示模板"下拉框中选择
3. 点击"提交"保存
4. 重新打开选片页面生效

### Q5: 数据多久刷新一次？

**A:** 
- 所有人像组循环展示完毕后，等待30秒自动刷新
- 如需立即刷新，可以刷新浏览器页面（F5）

## 扩展开发

### 新增模板

1. 在`templates/`目录下创建新模板文件夹
2. 创建`index.html`并实现模板逻辑

### 自定义配置

在模板的Vue实例中修改`config`对象：

```javascript
config: {
    imageDuration: 1000,      // 单张图片展示时长
    groupDuration: 5000,      // 单组展示时长
    refreshInterval: 30000,   // 数据刷新间隔
    avatarLimit: 15,          // 头像显示数量
    retryCount: 3,            // 请求重试次数
    retryDelay: 5000          // 重试间隔
}
```
