# 更新日志

## v1.2.0 (2026-01-23) - 头像点击切换

### ✨ 新增功能

- **头像点击切换**：用户可以点击下方头像列表，直接跳转到对应人像的照片
  - 鼠标悬停视觉反馈：头像放大1.1倍，边框变红
  - 平滑切换动画：500ms过渡效果
  - 智能播放控制：点击后暂停，500ms后恢复自动播放
  - 同步更新：头像高亮、二维码、播放状态同步更新

- **交互增强**：
  - 头像光标变为手型（cursor: pointer）
  - 悬停缩放效果
  - 点击区域优化（80px直径）

### 🔧 技术实现

- 新增 `switchToGroup(groupIndex)` 方法
- 增强 `renderAvatars()` 方法，添加点击事件
- 优化 CSS 样式，添加悬停效果
- 代码变更：+43行

### 🎯 适用场景

1. **游客主动选择**：点击感兴趣的头像查看照片
2. **门店人员引导**：快速定位到目标人像展示
3. **快速回看**：点击之前的头像重新播放

### 📝 文档更新

- 新增 `AVATAR_CLICK_FEATURE.md` 头像点击功能详细说明

## v1.1.0 (2026-01-22) - 演示数据支持

### ✨ 新增功能

- **演示数据自动展示**：当没有真实数据时，系统自动使用演示数据进行展示
  - 5组演示人像数据
  - 11张AI生成照片示例
  - 完整的轮播、头像、二维码功能演示

- **智能降级策略**：
  - 数据为空时自动启用演示模式
  - 网络请求失败重试3次后启用演示模式
  - 60秒后自动重试获取真实数据

- **演示模式提示**：
  - 页面顶部红色提示条
  - 5秒后自动消失
  - 友好的用户提示

- **演示预览页面**：
  - 新增 `demo.html` 演示引导页
  - 可视化功能说明
  - 一键预览演示效果

### 📝 文档更新

- 新增 `DEMO_DATA.md` 演示数据详细说明文档
- 更新 `README.md` 添加演示数据相关说明
- 更新 `QUICKSTART.md` 添加演示模式使用说明

### 🎯 适用场景

1. **商户预览**：配置门店时快速预览选片效果
2. **功能演示**：向客户展示系统功能无需等待数据
3. **开发测试**：前端独立开发，无需依赖后端真实数据

### 📊 演示数据详情
