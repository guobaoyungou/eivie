# AI旅拍选片端 - 快速启动指南

## 🚀 5分钟快速上手

### 第一步：登录后台

访问商家后台并登录：
```
后台地址 > 登录 > 进入管理系统
```

### 第二步：配置门店

1. 进入菜单：`系统` > `门店管理`
2. 点击任意门店的"编辑"按钮
3. 在编辑页面找到新增的两个字段：
   - **选片URL**：自动生成的专属链接
   - **展示模板**：选择喜欢的展示风格

### 第三步：获取链接

在门店编辑页面：

- **复制链接**：点击"复制链接"按钮，链接已复制到剪贴板
- **预览效果**：点击"预览"按钮，新窗口查看实际效果
- **获取二维码**：点击"二维码"按钮，弹窗显示可扫描的二维码

**提示：**即使还没有上传真实人像照片，也可以预览效果！系统会自动使用演示数据展示。

### 第四步：部署到大屏

1. 在门店大屏设备的浏览器中粘贴URL
2. 按F11进入全屏模式
3. 系统自动开始展示照片和视频

**建议浏览器：** Chrome 或 Edge（最新版本）

## ✨ 核心功能

### 1. 自动轮播展示
- 照片：每张展示1秒
- 组切换：每组展示5秒
- 视频：自动播放，静音循环

### 2. 智能刷新
- 自动刷新：所有内容展示完毕后30秒自动获取新数据
- 手动刷新：按F5键立即刷新

### 3. 二维码引导
- 实时显示：当前人像的二维码浮动在右下角
- 游客扫码：查看详情并购买

### 4. 多种模板
- Template 1：经典上下布局（默认）
- Template 2-5：其他风格（待开发）

## 📋 URL参数说明

完整URL格式：
```
https://域名/xpd/index.html?aid=1&bid=123&mdid=456
```

| 参数 | 必填 | 说明 | 示例 |
|-----|------|------|------|
| aid | 是 | 平台ID | 1 |
| bid | 是 | 商家ID | 123 |
| mdid | 否 | 门店ID | 456 |

**注意：** mdid不填则展示该商家所有门店的照片

## 🔧 常见操作

1. 后台 > 门店管理 > 编辑门店
2. "展示模板"下拉框选择新模板
3. 点击"提交"保存

## ❓ 常见问题

### Q: 页面显示"暂无内容"？
**A:** 可能原因：
- 该门店尚未上传人像照片
- 上传的照片还在生成中（等待5-30秒）
- 网络连接异常

**解决方案：**
1. 确认已上传人像照片
2. 等待AI生成完成
3. 检查网络连接
4. 刷新页面（F5）
5. **新：**系统会自动使用演示数据展示效果

### Q: 什么是演示模式？
**A:** 演示模式是系统内置的示例数据展示功能：
- 当没有真实数据时自动启用
- 包含5组人像和11张AI照片
- 页面顶部显示红色提示条
- 60秒后自动尝试获取真实数据
- 适用于预览和测试场景
- 详细说明：`/www/wwwroot/eivie/xpd/DEMO_DATA.md`

### Q: 轮播不流畅，卡顿？
**A:** 优化建议：
- 使用Chrome或Edge最新版浏览器
- 确保网络带宽充足（建议100M以上）
- 关闭其他占用资源的程序
- 检查设备性能（建议使用较新的设备）

### Q: 二维码不显示？
**A:** 可能原因：
- 该人像尚未生成二维码
- 二维码已过期
- 等待切换到下一组时重新检查

### Q: 如何知道当前展示的是哪个人像？
**A:** 下方头像列表中，中间位置高亮显示的就是当前人像

### Q: 可以同时在多个大屏上展示吗？
**A:** 可以！每个大屏都打开同一个URL即可，数据独立获取

## 🎯 最佳实践

### 门店部署建议

1. **设备选择**
   - 屏幕：43寸以上智能电视或显示器
   - 系统：Windows 10+ 或 Android TV
   - 浏览器：Chrome 最新版

2. **网络环境**
   - 带宽：100M以上
   - 稳定性：建议有线连接
   - 备份：准备4G流量卡作为备用

3. **日常维护**
   - 每天检查设备运行状态
   - 每周重启一次设备
   - 定期更新浏览器版本

### 内容运营建议

1. **照片质量**
   - 上传清晰的人像照片
   - 光线充足，背景简洁
   - 人脸清晰可见

2. **更新频率**
   - 旺季：每小时更新
   - 淡季：每天更新
   - 保持内容新鲜度

3. **吸引客流**
   - 在门店显眼位置放置大屏
   - 配合宣传海报引导游客关注
   - 提示游客扫码查看和购买
