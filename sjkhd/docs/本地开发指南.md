# 本地开发快速指南

## 📦 环境准备

### 1. 系统要求
- **Node.js**: v16.x 或 v18.x
- **操作系统**: Windows 10+, macOS 10.13+, 或 Linux（带图形界面）
- **编辑器**: VS Code（推荐）

### 2. 安装 Node.js
访问 https://nodejs.org/ 下载并安装 LTS 版本。

验证安装：
```bash
node --version
npm --version
```

---

## 🚀 快速开始

### 步骤一：下载项目到本地

#### 方法 A：使用 Git（推荐）
```bash
git clone <仓库地址>
cd sjkhd
```

#### 方法 B：从服务器下载

**Linux/macOS:**
```bash
scp -r root@服务器IP:/www/wwwroot/eivie/sjkhd ~/projects/
cd ~/projects/sjkhd
```

**Windows:**
- 使用 WinSCP (https://winscp.net/) 或 FileZilla
- 连接服务器后下载整个 sjkhd 目录

### 步骤二：安装依赖

```bash
# 进入项目目录
cd sjkhd

# 删除服务器的 node_modules（如果有）
rm -rf node_modules  # Windows: rmdir /s /q node_modules

# 配置国内镜像（可选，国内用户推荐）
npm config set registry https://registry.npmmirror.com

# 安装依赖
npm install
```

### 步骤三：启动开发服务器

```bash
npm run dev
```

**预期效果：**
- ✅ Vite 开发服务器启动（http://localhost:5173）
- ✅ Electron 窗口自动打开
- ✅ 显示应用界面
- ✅ 代码修改后自动热重载

## 📝 常用开发命令

### 启动开发
```bash
npm run dev                 # 启动开发模式（推荐）
```

### 分别启动各进程
```bash
npm run watch:main          # 监听主进程
npm run watch:preload       # 监听预加载脚本
npm run watch:renderer      # 监听渲染进程
npm start                   # 启动 Electron
```

### 代码检查
```bash
npm run lint                # 运行 ESLint
npm run lint:fix            # 自动修复 ESLint 问题
npm run type-check          # TypeScript 类型检查
```

### 构建项目
```bash
npm run build               # 完整构建
npm run build:main          # 仅构建主进程
npm run build:preload       # 仅构建预加载脚本
npm run build:renderer      # 仅构建渲染进程
```

### 打包应用
```bash
npm run dist                # 打包当前平台
npm run dist:win            # 打包 Windows 版本
npm run dist:mac            # 打包 macOS 版本
npm run dist:linux          # 打包 Linux 版本
```

## 🔧 开发技巧

### 1. 热重载
- 修改渲染进程代码（Vue 组件）会自动热重载
- 修改主进程代码需要重启 Electron（自动）

### 2. 调试主进程
- 在 VS Code 中按 F5 启动调试
- 或在终端查看日志输出

### 3. 调试渲染进程
- Electron 窗口会自动打开 Chrome DevTools
- 或按 `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (macOS)

### 4. 查看日志
日志文件位置：
- **Windows**: `%USERPROFILE%\AppData\Roaming\ai-travel-client-electron\logs\`
- **macOS**: `~/Library/Logs/ai-travel-client-electron/`
- **Linux**: `~/.config/ai-travel-client-electron/logs/`

## ⚠️ 常见问题

### 1. 端口被占用
```bash
# 修改 vite.config.ts 中的端口
server: {
  port: 5174  // 改为其他端口
}
```

### 2. Electron 窗口无法打开
- 检查是否有防火墙或安全软件阻止
- 尝试以管理员身份运行

### 3. 依赖安装失败
```bash
# 清理缓存重新安装
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### 4. TypeScript 报错
```bash
# 重新生成类型定义
npm run type-check
```
