# 本地开发环境配置清单

## ✅ 准备工作检查清单

### 第一步：下载项目到本地

- [ ] **方式 A：使用 SCP/SFTP 下载**
  ```bash
  # Linux/macOS
  scp -r root@服务器IP:/www/wwwroot/eivie/sjkhd ~/projects/
  
  # Windows: 使用 WinSCP 或 FileZilla
  # 1. 下载 WinSCP: https://winscp.net/
  # 2. 连接服务器
  # 3. 下载 /www/wwwroot/eivie/sjkhd/ 到本地
  ```

- [ ] **方式 B：使用 Git 克隆**（如果已配置）
  ```bash
  git clone <仓库地址>
  cd sjkhd
  ```

### 第二步：安装 Node.js（如果未安装）

- [ ] 访问 https://nodejs.org/ 下载 LTS 版本
- [ ] 安装完成后验证：
  ```bash
  node --version  # 应显示 v16.x 或 v18.x
  npm --version   # 应显示 8.x 或更高
  ```

### 第三步：配置项目

- [ ] 打开终端/命令行
- [ ] 进入项目目录
  ```bash
  cd ~/projects/sjkhd  # 替换为你的实际路径
  ```

- [ ] 删除服务器的 node_modules（如果一起下载了）
  ```bash
  # Linux/macOS
  rm -rf node_modules
  
  # Windows (PowerShell)
  Remove-Item -Recurse -Force node_modules
  
  # Windows (CMD)
  rmdir /s /q node_modules
  ```

- [ ] 配置国内镜像（可选，国内用户推荐）
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```

- [ ] 安装依赖
  ```bash
  npm install
  ```
  
  **预期结果**：
  - 创建 node_modules 目录
  - 显示安装进度
  - 最后显示 "added XXX packages"

### 第四步：启动开发服务器

- [ ] **方式 A：使用快捷脚本（推荐）**
  ```bash
  # Windows: 双击 start-dev.bat
  # 或在 CMD 中运行: start-dev.bat
  
  # macOS/Linux
  ./start-dev.sh
  ```

- [ ] **方式 B：使用 npm 命令**
  ```bash
  npm run dev
  ```

- [ ] **验证启动成功**
  - [ ] 终端显示 Vite 服务器启动信息
  - [ ] 显示 "http://localhost:5173"
  - [ ] Electron 窗口自动打开
  - [ ] 看到应用界面

### 第五步：配置 VS Code（推荐）

- [ ] 安装 VS Code: https://code.visualstudio.com/
- [ ] 打开项目
  ```bash
  code .  # 在项目目录运行
  ```
- [ ] 安装推荐扩展（会自动提示）
  - [ ] Vue Language Features (Volar)
  - [ ] TypeScript Vue Plugin (Volar)
  - [ ] ESLint
  - [ ] Prettier

- [ ] 验证扩展工作
  - [ ] 打开 `.vue` 文件有语法高亮
  - [ ] 打开 `.ts` 文件有类型提示
  - [ ] 保存文件时自动格式化

## 🎯 开发流程

### 日常开发步骤

1. **启动项目**
   ```bash
   npm run dev
   ```

2. **修改代码**
   - 编辑 `renderer/` 目录下的 Vue 组件
   - 编辑 `main/` 目录下的主进程代码
   - 保存后自动热重载

3. **调试代码**
   - 渲染进程：在 Electron 窗口中按 F12 打开 DevTools
   - 主进程：在 VS Code 中按 F5 启动调试

4. **提交代码**（如果使用 Git）
   ```bash
   git add .
   git commit -m "描述你的修改"
   git push
   ```

### 代码检查

在提交前运行：
```bash
npm run lint        # 检查代码规范
npm run type-check  # 检查类型错误
```

## ⚠️ 常见问题解决

### 问题 1：npm install 失败

**症状**：安装依赖时报错

**解决方案**：
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules 和 lock 文件
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### 问题 2：Electron 窗口无法打开

**症状**：npm run dev 运行但窗口不出现

**解决方案**：
1. 检查防火墙/安全软件是否阻止
2. 查看终端是否有错误信息
3. 尝试单独启动：
   ```bash
   npm run watch:renderer  # 终端1
   npm start              # 终端2
   ```

### 问题 3：端口被占用

**症状**：Vite 提示端口 5173 被占用

**解决方案**：
编辑 `vite.config.ts`：
```typescript
server: {
  port: 5174  // 改为其他端口
}
```

### 问题 4：TypeScript 报错

**症状**：编辑器显示大量红色波浪线

**解决方案**：
1. 重启 VS Code
2. 运行类型检查：
   ```bash
   npm run type-check
   ```
3. 确保安装了 Volar 扩展

### 问题 5：热重载不工作

**症状**：修改代码后页面不更新

**解决方案**：
1. 检查文件是否保存
2. 重启开发服务器（Ctrl+C 然后 npm run dev）
3. 清理浏览器缓存（在 DevTools 中右键刷新按钮 → 清空缓存并硬性重新加载）
