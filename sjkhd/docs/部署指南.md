# AI旅拍商家客户端 - 部署指南

## 打包前准备

### 1. 更新版本号
编辑 `package.json` 中的 `version` 字段。

### 2. 配置环境变量
创建 `.env` 文件，配置生产环境参数：
```env
API_BASE_URL=https://your-production-api.com
NODE_ENV=production
```

### 3. 准备应用图标
在 `resources/` 目录放置以下文件：
- `icon.png` (512x512)
- `icon.ico` (Windows)
- `icon.icns` (macOS)

## 打包命令

### 打包所有平台
```bash
npm run dist
```

### 仅打包 Windows
```bash
npm run dist:win
```

### 仅打包 macOS
```bash
npm run dist:mac
```

### 仅打包 Linux
```bash
npm run dist:linux
```

## 平台特定注意事项

### Windows
- 需要在 Windows 系统上打包
- 支持 x64 和 x86 架构
- 生成 NSIS 安装程序

### macOS
- 需要在 macOS 系统上打包
- 支持 Intel 和 Apple Silicon
- 生成 DMG 镜像文件
- 需要代码签名（生产环境）

### Linux
- 建议在 Linux 系统上打包
- 生成 AppImage 和 deb 包
- 仅支持 x64 架构

## 代码签名

### Windows
需要购买代码签名证书。

### macOS
需要 Apple Developer 账号和证书。

## 自动更新

配置 `electron-builder.json` 中的 `publish` 字段。

## 分发

### 直接下载
将安装包上传至文件服务器。

### 应用商店
- Windows: Microsoft Store
- macOS: Mac App Store
