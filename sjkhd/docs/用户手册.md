# AI旅拍商家客户端 - 用户手册

## 安装

### Windows
1. 下载安装程序
2. 双击运行
3. 按照向导完成安装

### macOS
1. 下载 DMG 文件
2. 打开并拖拽到应用程序文件夹
3. 首次运行需要在安全设置中允许

### Linux
1. 下载 AppImage 或 deb 包
2. 添加执行权限或安装

## 首次使用

### 1. 设备注册
- 启动应用
- 输入设备码（由管理员提供）
- 输入设备名称
- 选择商家和门店
- 点击注册

### 2. 配置监控文件夹
- 进入设置页面
- 添加需要监控的文件夹
- 配置文件过滤规则

### 3. 启动监控
- 返回首页
- 点击"启动监控"按钮
- 应用将自动检测新照片并上传

## 功能说明

### 首页
- 查看设备在线状态
- 查看监控状态
- 查看上传统计
- 快速操作按钮

### 上传列表
- 查看所有上传任务
- 查看任务进度
- 暂停/恢复上传
- 查看上传历史

### 设置
- 设备设置
- 监控设置
- 上传设置
- 应用设置

### 日志
- 查看运行日志
- 筛选日志级别
- 导出日志文件

## 常见问题

### 无法注册设备
- 检查网络连接
- 确认设备码正确
- 联系管理员

### 文件未被检测
- 确认监控已启动
- 检查文件格式是否支持
- 查看日志了解详情

### 上传失败
- 检查网络连接
- 查看错误信息
- 尝试重新上传

## 系统托盘

应用最小化后会在系统托盘运行，右键托盘图标可以：
- 显示/隐藏主窗口
- 启动/停止监控
- 查看队列状态
- 退出应用
