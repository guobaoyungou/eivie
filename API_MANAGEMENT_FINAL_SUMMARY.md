## 技术亮点

### 1. 反射机制应用
使用PHP反射（ReflectionClass、ReflectionMethod）实现控制器自动扫描，无需手动维护接口列表。

### 2. 智能分类识别
根据控制器命名规则自动识别接口分类，支持自定义扩展。

### 3. 在线HTTP测试
内置HTTP客户端，支持GET/POST等多种请求方式，实时查看响应结果。

### 4. JSON数据处理
完善的JSON编解码、格式化、语法高亮显示功能。

### 5. 日志审计机制
记录所有测试操作，包括请求参数、响应数据、响应时间等，便于问题追踪。

## 部署指南

### 快速部署（3步）

#### 第1步：导入数据库
```bash
mysql -u用户名 -p密码 数据库名 < /www/wwwroot/eivie/database/migrations/api_management_tables.sql
mysql -u用户名 -p密码 数据库名 < /www/wwwroot/eivie/database/migrations/api_auth_interfaces_data.sql
```

#### 第2步：清理缓存
```bash
cd /www/wwwroot/eivie
rm -rf runtime/cache/*
```

#### 第3步：登录测试
1. 使用平台管理员账号登录
2. 查看"API"菜单
3. 点击"接口扫描"开始使用

## 使用流程

### 首次使用
```
1. 登录后台（平台管理员）
   ↓
2. 点击"API > 接口扫描"
   ↓
3. 选择控制器，开始扫描
   ↓
4. 查看扫描结果，保存接口
   ↓
5. 返回"接口列表"查看所有接口
```

### 日常使用
```
1. 查看接口列表
   ↓
2. 搜索或筛选需要的接口
   ↓
3. 点击接口查看详情
   ↓
4. 点击"在线测试"进行测试
   ↓
5. 查看"测试历史"了解测试记录
```
