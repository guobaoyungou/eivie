# 照片/视频生成页使用指南

## 🚀 快速开始

### 访问页面

在浏览器中访问以下URL：

- **照片生成页**: `http://your-domain/?s=/Index/photo_generation`
- **视频生成页**: `http://your-domain/?s=/Index/video_generation`

或通过导航菜单访问：
1. 点击左侧边栏"创作"菜单
2. 选择"图片生成"或"视频生成"

## 📋 功能说明

### 照片生成页

#### 生成方式（Tab切换）

1. **描述生成** 📝
   - 输入文字描述或提示词
   - 选择AI模型
   - 设置图片尺寸（1:1, 3:4, 4:3, 16:9, 9:16）
   - 选择生成张数（1-9张）
   - 可开启随机模式

2. **图片生成** 🖼️
   - 上传参考图片
   - 输入调整描述
   - 选择AI模型

3. **数字人** 👤
   - 输入数字人相关描述
   - 生成虚拟人物形象

4. **3D模型** 🎭
   - 输入3D模型描述
   - 生成三维模型

### 视频生成页

#### 生成方式（Tab切换）

1. **描述生成** 📝
   - 输入视频描述或提示词
   - 选择AI模型
   - 设置视频比例（16:9, 9:16, 1:1）
   - 选择视频时长（5s, 10s, 15s, 30s）
   - 可开启随机模式

2. **图片** 🖼️
   - 上传图片
   - 输入视频描述
   - 将图片转换为视频

3. **视频** 🎬
   - 上传视频
   - 输入调整描述
   - 对视频进行AI处理

## 💡 右侧推荐模板

## ⚙️ 管理员配置

### 前置条件

1. **启用模板三**
   - 后台设置 → 网站信息
   - 设置 `showweb = 3`

2. **准备模板数据**
   ```sql
   -- 确保有场景模板数据
   SELECT * FROM generation_scene_template 
   WHERE status = 1 
   ORDER BY use_count DESC;
   ```

3. **检查模板封面**
   - 确保 `cover_image` 字段有有效图片路径
   - 图片建议尺寸：4:3 比例

## 🐛 常见问题

### Q1: 页面显示404
**A**: 检查以下内容：
- 确保 `showweb = 3`（模板三）
- 清除路由缓存
- 检查Index控制器是否有对应方法

### Q2: 推荐模板区域为空
**A**: 
- 检查数据库是否有模板数据
- 确认 `generation_type` 字段正确（1=照片，2=视频）
- 确认 `status = 1`（启用状态）

### Q3: 样式显示异常
**A**:
- 清除浏览器缓存
- 检查CSS文件路径是否正确
- 查看浏览器控制台是否有404错误

### Q4: 点击生成按钮无反应
**A**:
- 打开浏览器控制台查看错误
- 检查是否填写必填字段
- 确认generation.js已正确加载
