#### 修改2：添加关闭订单接口

新增方法 `closeOrder()`：

```php
/**
 * 关闭订单
 */
public function closeOrder()
{
    $orderId = input('param.id', 0);
    
    // 查询选片订单
    $order = Db::name('ai_travel_photo_order')
        ->where('id', $orderId)
        ->where('aid', aid)
        ->where('uid', mid)
        ->find();
    
    if ($order) {
        // 只有待付款订单才能关闭
        if ($order['status'] != 0) {
            return $this->json(['status' => 0, 'msg' => '只有待付款订单才能关闭']);
        }
        
        // 关闭订单
        Db::name('ai_travel_photo_order')
            ->where('id', $orderId)
            ->update([
                'status' => 3,  // 3=已关闭
                'close_time' => time(),
                'update_time' => time()
            ]);
        
        // 关闭对应的支付订单
        Db::name('payorder')
            ->where('aid', aid)
            ->where('type', 'ai_pick')
            ->where('orderid', $orderId)
            ->where('status', 0)
            ->update(['status' => -1]);
        
        return $this->json(['status' => 1, 'msg' => '订单已关闭']);
    }
    
    return $this->json(['status' => 0, 'msg' => '订单不存在']);
}
```

#### 修改3：订单详情返回 payorderid

修改 `getAiPickOrderDetail()` 方法，添加支付订单ID查询：

```php
protected function getAiPickOrderDetail($order)
{
    // ... 其他代码
    
    $detail = [
        'id' => $order['id'],
        'ordernum' => $order['order_no'] ?? '',
        'status' => $order['status'] ?? 0,
        // ... 其他字段
        'payorderid' => $this->getPayOrderId($order['id'], $order['status'] ?? 0),  // ← 新增
    ];
    
    return $detail;
}

/**
 * 获取支付订单ID
 */
protected function getPayOrderId($orderId, $status)
{
    if ($status != 0) {
        return 0;
    }
    
    $payorder = Db::name('payorder')
        ->where('aid', aid)
        ->where('type', 'ai_pick')
        ->where('orderid', $orderId)
        ->where('status', 0)
        ->value('id');
    
    return $payorder ? (int)$payorder : 0;
}
```

## 注意事项

1. **权限校验**：关闭订单时会校验 `uid=mid`，确保用户只能关闭自己的订单
2. **状态限制**：只有 `status=0`（待付款）的订单才能关闭
3. **级联操作**：关闭订单时会同时关闭对应的支付订单（`payorder.status=-1`）
4. **支付订单ID**：只有待付款订单才返回 `payorderid`，已付款订单返回 `0`
5. **页面路由**：需要在 `pages.json` 中注册新页面路由
