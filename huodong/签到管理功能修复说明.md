## 🎯 新增功能使用说明

### 1. 权限管理界面
在"签到名单"页面中，现在可以看到：

#### 新增列：
1. **权限状态列**：显示用户权限状态
   - `管理员` - 绿色徽章
   - `核销员` - 蓝色徽章
   - `无权限` - 灰色文字

2. **权限管理列**：提供操作按钮
   - **管理员按钮**：绿色表示已是管理员，灰色表示非管理员
   - **核销员按钮**：蓝色表示已是核销员，灰色表示非核销员
   - **加入白名单按钮**：原有功能保留

#### 新增按钮：
- **批量权限管理按钮**：在工具栏添加"权限管理"按钮（蓝色）
  - 可批量设置管理员或核销员权限
  - 支持添加或移除权限

### 2. 数据库结构
- **新增字段**：
  `ddwx_hd_participant` 表添加：
  - `is_admin` TINYINT(1) DEFAULT 0 - 是否管理员
  - `is_verifier` TINYINT(1) DEFAULT 0 - 是否核销员

### 3. API接口
- **权限管理API**：`/huodong/api/sign/permissions.php`
  - `GET` - 获取权限列表（支持筛选）
  - `POST` - 设置单个用户权限
  - `PUT` - 批量设置用户权限

## 🚀 使用步骤

1. **检查数据库表结构**：
   ```
   http://your-domain/huodong/api/sign/check_table.php (仅限本地访问)
   ```

2. **进入签到管理**：
   - 选择活动 → 签到管理 → 签到名单

3. **查看权限状态**：
   - 在签到名单表格中查看权限状态列

4. **管理权限**：
   - **单个用户**：点击表格中的"管理员"或"核销员"按钮
   - **批量管理**：勾选多个用户 → 点击工具栏"权限管理"按钮
