## ⚠️ 已知限制

### 当前版本限制

1. **队列持久化** ⏳
   - 问题：重启后队列清空
   - 影响：未完成的上传任务丢失
   - 解决方案：集成SQLite存储队列状态

2. **断点续传** ⏳
   - 问题：不支持断点续传
   - 影响：大文件上传失败需重新传输
   - 解决方案：实现分片上传和断点续传

3. **大文件支持** ⏳
   - 问题：默认限制10MB
   - 影响：大文件无法上传
   - 解决方案：实现分片上传

4. **单实例检测** ⏳
   - 问题：可能启动多个客户端实例
   - 影响：重复上传、资源浪费
   - 解决方案：实现Mutex单实例检测
