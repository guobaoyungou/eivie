## 一、产品介绍

### 1.1 产品概述

AI旅拍商家Windows客户端是一款专为摄影门店设计的照片自动上传工具。它能够自动监控指定文件夹,当检测到新增的照片时,自动上传到云端服务器,无需手动操作,大大提高工作效率。

### 1.2 主要功能

- **自动监控**: 实时监控文件夹,自动检测新照片
- **智能上传**: 自动上传照片到服务器
- **断网续传**: 网络恢复后自动继续上传
- **重复过滤**: 自动过滤重复照片,避免浪费流量
- **实时统计**: 显示上传成功/失败数量
- **操作日志**: 记录所有操作,方便追踪

### 1.3 使用场景

- 旅游景区摄影门店
- 主题乐园拍照点
- 婚纱摄影工作室
- 连锁摄影门店

### 1.4 系统要求

- Windows 7及以上操作系统
- 稳定的网络连接
- 至少2GB内存
- 100MB硬盘空间

## 二、安装指南

### 2.1 下载安装包

从管理员处获取安装包 `AiTravelClient_Setup.exe`

### 2.2 安装步骤

1. **运行安装程序**
   - 右键点击安装包
   - 选择"以管理员身份运行"

2. **选择安装路径**
   - 默认安装到: C:\Program Files\AiTravelClient
   - 可以点击"浏览"选择其他位置

3. **确认安装选项**
   - 勾选"创建桌面快捷方式"(推荐)
   - 勾选"开机自动启动"(可选)

4. **开始安装**
   - 点击"安装"按钮
   - 等待安装完成

5. **完成安装**
   - 点击"完成"按钮
   - 双击桌面图标启动程序

## 三、快速入门

### 3.1 首次启动

第一次启动程序时,会弹出设备注册向导。

### 3.2 设备注册

**步骤1: 输入服务器地址**
```
服务器地址: https://your-domain.com
```
- 向管理员获取正确的服务器地址
- 点击"测试连接"验证地址是否正确
- 显示"连接成功"后点击"下一步"

**步骤2: 输入设备信息**
```
设备编码: _________ (向管理员获取)
商家ID: ___________
设备名称: 摄影门店A号机
```
- 设备编码需要从管理后台获取
- 设备名称可以自定义,便于识别

**步骤3: 完成注册**
- 点击"注册设备"按钮
- 等待系统验证
- 显示"注册成功"后进入主界面

### 3.3 添加监控文件夹

**步骤1: 打开设置**
- 点击顶部的"设置"选项卡

**步骤2: 添加文件夹**
- 在"监控设置"区域点击"添加文件夹"
- 浏览并选择需要监控的文件夹
- 例如: D:\照片\待上传

**步骤3: 保存配置**
- 可以添加多个文件夹
- 点击"保存配置"按钮

### 3.4 开始监控

- 返回"首页"选项卡
- 点击"开始监控"按钮
- 状态显示"监听中"
- 现在可以开始使用了!

- 自动上传: 检测到文件后是否自动上传
- 并发上传数: 同时上传几个文件(建议3个)
- 重试次数: 失败后重试几次


## 五、常见问题

### Q1: 照片没有被自动上传?

**可能原因:**
1. 监控功能未启动
2. 文件类型不支持(只支持JPG/PNG)
3. 文件太大或太小(要求10KB-10MB)
4. 网络连接异常

**解决方法:**
1. 检查主界面状态是否显示"监听中"
2. 确认文件格式为JPG或PNG
3. 检查文件大小是否符合要求
4. 检查网络连接是否正常

### Q2: 上传失败怎么办?

**解决方法:**
1. 查看日志了解失败原因
2. 检查网络连接
3. 等待自动重试(系统会自动重试5次)
4. 如果一直失败,联系技术支持

### Q3: 如何查看上传记录?

**方法:**
1. 在主界面查看实时日志
2. 点击"日志"选项卡查看历史日志
3. 可以按日期和关键字搜索
4. 可以导出日志文件保存

### Q4: 重复的照片会上传多次吗?

**回答:**
不会。系统会自动计算文件的MD5值,如果检测到重复文件,会自动跳过,避免浪费流量。

### Q5: 可以同时监控多个文件夹吗?

**回答:**
可以。在设置页面可以添加多个监控文件夹,系统会同时监控所有文件夹。

### Q6: 如何暂停上传?

**方法:**
点击主界面的"暂停监控"按钮,系统会停止监控和上传。需要恢复时再次点击"开始监控"。

### Q7: 网络断开后会丢失照片吗?

**回答:**
不会。网络断开后,照片会保留在上传队列中。网络恢复后,系统会自动继续上传。

### Q8: 如何更换监控文件夹?

**方法:**
1. 进入"设置"页面
2. 在监控文件夹列表中点击[×]删除旧文件夹
3. 点击"添加文件夹"添加新文件夹
4. 点击"保存配置"

### Q9: 忘记设备编码怎么办?

**方法:**
联系管理员,在后台管理系统中查询设备编码。

### Q10: 如何卸载软件?

**方法:**
1. 打开Windows控制面板
2. 进入"程序和功能"
3. 找到"AI旅拍商家客户端"
4. 右键点击,选择"卸载"

## 六、故障排查

### 6.1 软件无法启动

**症状:** 双击图标后没有反应

**排查步骤:**
1. 检查是否已安装.NET Framework 4.7.2
2. 右键点击程序,选择"以管理员身份运行"
3. 检查Windows事件日志是否有错误信息
4. 重新安装软件

### 6.2 连接服务器失败

**症状:** 显示"连接服务器失败"

**排查步骤:**
1. 检查网络连接是否正常
2. 使用浏览器访问服务器地址测试
3. 检查防火墙是否阻止程序访问网络
4. 确认服务器地址是否正确
5. 联系技术支持

### 6.3 文件未被检测到

**症状:** 复制照片到监控文件夹,没有被检测到

**排查步骤:**
1. 确认监控状态是否为"监听中"
2. 检查文件类型是否为JPG或PNG
3. 检查文件大小是否在10KB-10MB范围内
4. 检查文件名是否以.或~开头(临时文件会被过滤)
5. 查看日志是否有相关信息

### 6.4 上传速度慢

**症状:** 照片上传很慢

**排查步骤:**
1. 检查网络带宽
2. 检查是否使用WiFi连接(建议使用有线网络)
3. 调整并发上传数(在设置中增加)
4. 检查服务器性能
5. 避免同时运行其他占用带宽的程序

### 6.5 程序占用资源高

**症状:** CPU或内存占用过高

**排查步骤:**
1. 检查监控文件夹中是否有大量文件
2. 减少并发上传数
3. 重启程序
4. 清理过期日志
5. 联系技术支持

### 6.6 设备显示离线

**症状:** 后台显示设备离线

**排查步骤:**
1. 检查程序是否正在运行
2. 检查网络连接
3. 查看日志中心跳是否成功
4. 重启程序
5. 重新注册设备

## 七、使用技巧

### 7.1 最佳实践

1. **使用有线网络**
   - 比WiFi更稳定可靠
   - 上传速度更快

2. **合理设置并发数**
   - 网速快可以设置3-5个并发
   - 网速慢建议设置1-2个并发

3. **定期查看日志**
   - 及时发现问题
   - 了解上传情况

4. **保持程序运行**
   - 设置开机自动启动
   - 最小化到系统托盘

5. **定期清理监控文件夹**
   - 及时转移已上传的照片
   - 避免文件夹文件过多

### 7.2 注意事项

1. **不要频繁修改配置**
   - 配置稳定后不建议频繁修改

2. **注意文件格式**
   - 只支持JPG和PNG格式
   - 不支持RAW等其他格式

3. **注意文件大小**
   - 单个文件不超过10MB
   - 过大的文件需要先压缩

4. **保护设备Token**
   - 不要向他人泄露config.json文件
   - Token丢失需要重新注册

5. **及时更新软件**
   - 关注新版本发布
   - 及时升级获取新功能
